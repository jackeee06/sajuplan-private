# 문의(QnA) — 상담사 1:1 문의 CRUD + 비즈룰

## 한 줄로 답하면
**회원이 상담사 프로필에서 1:1 문의를 작성하고, 상담사가 답변한다. 문의는 제3자에게 항상 비공개. 같은 상담사에게 하루 5개까지만 가능.**

## 누가 무엇을 하나

| 역할 | 할 수 있는 일 | 화면 |
|---|---|---|
| 회원(작성자) | 문의 작성 / 수정 / 삭제 / 신고 / 내 문의 모아보기 | 상담사 상세 → 문의 탭, 마이페이지 → 나의 상담문의 |
| 상담사(대상) | 내게 온 문의 확인 / 답변 작성·수정·삭제 | 상담사 마이페이지 → 고객 문의 |
| 제3자(다른 회원) | 목록의 제목/상태만 보임, **본문은 못 봄** | 상담사 상세 → 문의 탭 |

## 문의 작성 규칙 (검증된 비즈룰)

| 규칙 | 내용 | 어기면 |
|---|---|---|
| 제목/내용 필수 | 둘 다 비어 있으면 안 됨 | "제목을 입력해주세요." / "문의 내용을 입력해주세요." (400) |
| 제목 길이 | 최대 255자 | "제목이 너무 깁니다. (최대 255자)" (400) |
| 대상이 상담사여야 | role=counselor, 탈퇴 안 한 사람만 | "상담사를 찾을 수 없습니다." (404) |
| **본인 페이지 문의 금지** | 내가 나한테 문의 못 함 | "본인 페이지에는 문의할 수 없습니다." (403) |
| **하루 5개 제한** | 같은 상담사에게 1일 최대 5개 | "같은 상담사에게 하루 최대 5개까지 문의할 수 있습니다." (400) |

> 문의는 **항상 비밀글**로 저장된다. 작성 화면에 공개/비밀 토글이 없는 이유. 본문은 작성자 본인과 해당 상담사만 볼 수 있다.

## 조회(비공개) 정책

- **문의 본문은 제3자에게 항상 가려진다.** 목록에서 다른 사람 문의는 제목·날짜·답변여부만 보이고 내용은 빈칸.
- 작성자 본인 또는 해당 상담사 본인만 본문을 볼 수 있다.
- 신고가 누적돼 숨김 처리된 문의는 작성자 본인/상담사만 접근 가능, 나머지에겐 "문의를 찾을 수 없습니다."로 안 보인다.

## 수정·삭제 규칙

| 동작 | 조건 |
|---|---|
| 문의 수정 | 본인 작성 글만 + **답변이 달리기 전**까지만 (제목·내용만) |
| 문의 삭제 | 본인 작성 글만 + **답변이 달리기 전**까지만 |
| 답변 달린 후 | 수정·삭제 둘 다 불가 ("답변이 달린 문의는 수정/삭제할 수 없습니다.") |

> 마이페이지 "나의 상담문의" 목록에서 답변이 없는 문의에만 ⋮(더보기) 메뉴가 보인다. 답변 완료된 건은 수정/삭제 버튼이 사라진다.

## 신고

- 다른 회원이 부적절한 문의를 신고할 수 있다.
- **같은 문의에 신고 3건 누적 → 자동 숨김.** 이후 제3자에게 안 보인다.
- 신고가 들어오면 작성자에게 앱 푸시 알림이 간다.

## 알림(알림톡 + 푸시)

| 시점 | 받는 사람 | 채널 |
|---|---|---|
| 회원이 문의 작성 | 상담사 | 알림톡(qa_ask_v2) + FCM 푸시 |
| 상담사가 답변 등록 | 회원 | 알림톡(qa_answer_v2) + (앱 링크) |
| 내 문의가 신고됨 | 작성자 | FCM 푸시 |

> 알림톡 실패해도 문의 작성·답변 저장은 정상 처리된다(알림은 부가 기능).

## 2026-06-11 버그 수정 (운영자용 요약)

### "본인 페이지 문의 금지"가 안 먹혀서 self 문의 5건 발생
- 증상: 상담사가 자기 자신의 페이지에 문의를 5건이나 남길 수 있었음. 막혀 있어야 할 동작이 뚫림.
- 원인: 로그인 정보의 사용자번호가 내부적으로 "글자" 형태로 들어오는데, 차단 검사는 "숫자"로 비교하고 있었음. 형식이 달라 "같은 사람"인데도 "다른 사람"으로 판정 → 차단이 조용히 무력화됐음.
- 조치: 로그인 정보 사용자번호를 들어오는 즉시 숫자로 정규화하도록 근본 수정 + 문의 차단 검사도 양쪽을 숫자로 맞춰 이중 안전. 이제 본인 페이지 문의는 정상 차단됨.
- 같은 뿌리의 문제라 문의(qna) 외 상담(consult)·상담사(counselors) 소유권 검사도 함께 안전해졌음.

## 이건 정상인가요? 에러인가요?

| 상황 | 답변 |
|---|---|
| 다른 회원 문의 본문이 안 보임 | ✅ 정상 (문의는 항상 비공개) |
| 같은 상담사에게 6번째 문의 시도 → 막힘 | ✅ 정상 (하루 5개 제한) |
| 답변 달린 문의 수정 버튼 사라짐 | ✅ 정상 |
| 상담사가 자기 페이지에 문의 가능 | ❌ 비정상 (2026-06-11 차단됨. 발생 시 보고) |
| 문의 신고 3건 → 자동 숨김 | ✅ 정상 |
