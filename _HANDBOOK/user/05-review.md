# 후기(리뷰) — 작성·수정·삭제·코인

## 한 줄로 답하면
**회원은 "5분 이상 진행한 본인 상담"에만 후기를 1건 쓸 수 있다. 후기를 쓰면 코인이 적립되고(일반 500 / 사진 +500), 작성 5분 안에만 수정·삭제가 가능하다. 후기는 항상 공개다.**

## 후기를 쓸 수 있는 조건 (작성)

| 조건 | 값 | 안 맞으면 |
|---|---|---|
| 어떤 상담의 후기인지(상담번호) | **필수** | "상담 내역이 있어야 후기를 작성할 수 있습니다" |
| 내 상담인가 | 본인 소유만 | "본인의 상담만 후기를 작성할 수 있습니다" |
| 상담이 끝났는가 | 종료된 상담만 | "종료된 상담에만 후기 작성이 가능합니다" |
| 얼마나 진행했나 | **5분(300초) 이상** | "5분 이상 상담을 진행한 경우에만…" |
| 언제까지 쓰나 | 상담 종료 후 **7일 이내** | "상담 종료 후 7일 이내에만…" |
| 같은 상담 중복 | 1건만 | "이미 후기를 작성한 상담입니다" |

> **5분 규칙의 이유**: 문제 손님이 상담 시작하자마자 끊고 악의적 후기를 쓰는 걸 막기 위한 상담사 보호 정책. 5분을 채우지 못한 상담은 후기 대상이 아니다.

후기는 **항상 공개**다. 비밀글 기능은 폐지됐다(예전 입력값이 남아 있어도 무시하고 공개로 저장).

## 수정 (작성 후 5분 + 상담사 답변 전)

| 조건 | 값 |
|---|---|
| 누가 | 작성한 본인만 |
| 언제까지 | 작성 후 **5분(300초) 이내** |
| 상담사가 답변을 달았으면 | 수정 불가 |
| 무엇을 | **제목·내용만** (별점·사진·공개여부는 못 바꿈) |

5분이 지났거나 상담사 답변이 달리면 "수정할 수 없습니다" 안내가 뜬다. 정상이다.

## 삭제

- 작성한 본인만 삭제 가능 (완전 삭제 — 복구 불가).
- 삭제하면 후기로 받았던 코인이 **자동 회수**된다.
- (수정과 동일하게 5분·상담사 답변 제약이 적용된다.)

## 후기 코인 정책 (현재값)

| 항목 | 값 |
|---|---|
| 일반 후기 작성 | 500코인 |
| 사진 첨부 | +500코인 (합계 1,000코인) |
| 베스트 후기 선정(관리자) | +10,000코인 (1회만, 멱등) |
| 삭제 시 | 받았던 코인 회수 |

> 코인 적립은 "후기 관리자 설정(payout_enabled)이 켜져 있고, 상담 사용액이 최소 기준 이상"일 때만 나간다. 적립이 실패해도 후기 자체는 정상 저장된다(코인은 best-effort).

## 사진 후기

- 모바일 카메라 원본 사진(최대 **30MB**)까지 첨부 가능.
- 서버가 자동으로 가볍게 변환(webp, 1024px)해서 저장 — 8.7MB 원본도 0.33MB로 줄어든다.
- 사진은 한 장만 등록.

## 베스트 후기

| 종류 | 누가 선정 | 한도 | 보상 |
|---|---|---|---|
| 상담사 베스트 | 후기를 받은 **상담사 본인** | 상담사당 최대 **5개** | 없음(노출 강조용) |
| 관리자 베스트 | 운영자 | — | 작성자에게 **10,000코인** (1회) |

- 상담사가 6번째 베스트를 누르면 "최대 5개" 안내 → 기존 해제 후 다시 시도.
- 관리자 베스트 코인은 같은 후기에 **단 1회**만 지급된다(중복 클릭해도 추가 지급 없음).

## 신고

- 다른 회원이 부적절한 후기를 신고할 수 있다.
- **본인 후기는 신고 불가**.
- 같은 후기를 같은 사람이 두 번 신고하면 "이미 신고하신 후기입니다".
- 신고는 운영자가 검토 후 처리 — **자동 숨김은 없다**(임계값 자동 숨김 미적용).

## 이건 정상인가요? 에러인가요?

| 상황 | 답변 |
|---|---|
| 5분 못 채운 상담에 후기 시도 → 막힘 | ✅ 정상 (상담사 보호) |
| 같은 상담에 후기 2번 시도 → 막힘 | ✅ 정상 |
| 작성 6분 뒤 수정 시도 → 막힘 | ✅ 정상 (5분 초과) |
| 상담사 답변 후 수정·삭제 막힘 | ✅ 정상 |
| 후기 삭제 → 코인 줄어듦 | ✅ 정상 (회수) |
| 30MB 카메라 사진 첨부 성공 | ✅ 정상 (2026-06-10부터) |
| 같은 상담 후기 2번에 코인 또 지급 | ❌ 비정상 (차단됨. 발생 시 보고) |
| 관리자 베스트 또 눌렀는데 10,000코인 또 지급 | ❌ 비정상 (1회 멱등. 발생 시 보고) |

## 이런 상황을 보셨나요?

### "후기를 쓰려는데 안 써진대요"
→ 십중팔구 5분 미만 상담이거나, 종료 7일이 지났거나, 이미 그 상담에 후기를 쓴 경우. 위 작성 조건표 확인.

### "후기를 수정하려는데 안 돼요"
→ 작성 5분이 지났거나 상담사가 이미 답변을 달았을 때. 둘 다 정상 차단.

### "후기 코인이 안 들어왔어요"
→ 후기 설정(payout_enabled)이 꺼져 있거나, 그 상담의 사용액이 최소 기준 미만일 수 있음. 코인 적립은 조건부.
