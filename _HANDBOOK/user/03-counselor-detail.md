# 상담사 상세 (탭 · 공지 · 후기 · 문의)

## 한 줄로 답하면
**회원이 상담사 카드를 누르면 들어가는 상세 페이지. 히어로 사진 + 프로필 + 약력 + 공지사항 위에, 아래쪽에 소개·후기·문의 3개 탭이 있다. 하단에는 단골·전화상담·채팅상담 버튼이 항상 고정.**

## 화면 구조 (위 → 아래)

```
① 히어로 사진 (가로 꽉, 5:4 비율) + 뒤로/홈 버튼
② 프로필 카드 — 카테고리 뱃지 / 이름 / 번호 / 한줄소개 / 해시태그 / 30초당 가격 / 공유 / 하트
③ 전문분야 · 스타일
④ 상담사 약력 (불릿)
⑤ 상담사 공지사항 (날짜 + 내용)
─── 회색 8px 구분선 ───
⑥ 탭 바 (소개 · 후기(건수) · 문의(건수))  ← 스크롤하면 화면 상단에 붙음(sticky)
⑦ 탭 본문 (선택한 탭만 표시)
⑧ 하단 고정 버튼 — 하트 / 전화상담 / 채팅상담 (항상 화면 맨 아래)
```

## 탭 동작

- 탭은 **소개 / 후기 / 문의** 3개.
- 탭을 누르면 본문만 바뀌고, **화면이 탭 위치로 스르륵 내려간다** (히어로 사진까지 안 올라감).
- 탭을 누르면 주소(URL)도 같이 바뀐다 — 소개는 기본 주소, 후기는 `?tab=reviews`, 문의는 `?tab=qna`.
  - 그래서 후기 탭 상태로 링크를 복사해 공유하면, 받은 사람도 바로 후기 탭이 열린다.
- 한 번 본 탭은 다시 와도 빠르다 (내용을 기억해 둠).

## 카테고리 뱃지 색

상담사 이름 앞 작은 알약 뱃지. 카테고리별 색이 정해져 있다.

| 카테고리 | 색 |
|---|---|
| 타로 | 핑크 |
| 신점 | 청록 |
| 사주 | 빨강 |
| 심리 | 보라 |

- 카테고리가 '기타'로 잡히면 기본값 '사주' 뱃지로 보인다.
- **전속파트너** 상담사는 옆에 검은 알약 뱃지(ⓘ)가 추가로 붙는다. ⓘ를 누르면 "오직 사주플랜에서만 상담" 설명이 뜬다.

## 공지사항이 깨져 보일 때 (2026-06-04 수정 완료)

> 예전에 공지사항에 `<p>`, `<br>` 같은 글자가 그대로 보이는 버그가 있었다. **지금은 수정됨.** 만약 또 보이면 캐시 의심(앱 재시작) 후 점검 요청.

- 공지사항은 상담사가 에디터로 꾸민 글이라 줄바꿈·굵게 등이 들어간다. 정상적으로 모양이 적용되어 보여야 한다.
- 혹시 공지에 광고/외부 링크 같은 위험한 코드가 들어가도 자동으로 걸러진다 (안전 처리).

## 단골(하트) 버튼

- 프로필 우측 하트, 하단 고정바 왼쪽 하트 **둘 다 같은 단골 버튼**. 누르면 핑크색으로 바뀌며 단골 등록.
- 비로그인 상태로 누르면 로그인 안내 모달이 뜬다.
- **본인을 단골 등록하려 하면 차단**된다 (상담사가 자기 상세를 봤을 때).
- 단골 수(하트 옆 숫자)는 실시간 집계. 999명 넘으면 `999+`로 표시.

## 전화상담 / 채팅상담 버튼

- 하단 고정바의 핑크 버튼 2개. 누르면 상담 시작 모달(시간·코인 선택)이 뜬다.
- 이 버튼은 항상 화면 맨 아래 고정이라 어느 탭에서나 바로 상담 시작 가능.

## 없는 상담사 페이지에 들어가면

- 탈퇴했거나 없는 번호로 들어가면 **"해당 상담사를 찾을 수 없습니다." + 뒤로 가기 버튼**이 뜬다. (빈 화면 아님)

## 이건 정상인가요? 에러인가요?

| 상황 | 답변 |
|---|---|
| 탭 클릭 → 본문 바뀌고 탭 위치로 스크롤 | ✅ 정상 |
| 탭 클릭 시 히어로 사진까지 올라감 | ❌ 버그(2026-06-04 수정됨). 재발 시 캐시 의심 |
| 공지사항에 `<p>` 같은 글자 노출 | ❌ 버그(수정됨). 재발 시 점검 |
| 하트 눌렀는데 로그인 모달 | ✅ 정상 (비로그인) |
| 상담사가 자기 상세에서 하트 눌렀는데 안 됨 | ✅ 정상 (본인 단골 차단) |
| 후기가 20개까지만 보이고 "더보기" 버튼 | ✅ 정상 (20개씩 끊어 로드) |
| 없는 번호 상세 → "찾을 수 없습니다" | ✅ 정상 |

## 회원이 자주 묻는 질문

### Q. "후기가 몇 개 없어 보여요"
후기 탭은 한 번에 20개만 불러온다. 아래 **"상담 후기 더보기"** 를 누르면 다음 20개가 이어 로드된다.

### Q. "공지 글씨체/줄이 이상해요"
상담사가 직접 작성한 서식이 그대로 보이는 것. 깨진 태그 글자가 보이면(`<p>` 등) 그건 버그이니 캡처 요청.

## 운영자가 확인하는 위치

| 확인 항목 | 위치 |
|---|---|
| 상담사 공지/소개 원문 | 관리자 → 상담사 관리 → 해당 상담사 (post_counselor) |
| 상담사 카테고리/전속 여부 | 관리자 → 상담사 관리 상세 |
| 후기/문의 건수 | 상세 페이지 탭 라벨에 자동 표시 |
