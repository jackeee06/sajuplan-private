# 내 상담 내역 (전화·채팅)

## 한 줄로 답하면
**회원이 받은 상담을 마이페이지에서 확인하는 화면. 전화상담 내역(`/mypage/calls`)·채팅상담 내역(`/mypage/chats`)·통합 상담내역(`/mypage/history`) 세 가지. 종료된 상담만 보이며, 상담사·시간·사용시간·사용 코인·후기 작성 여부가 함께 표시된다.**

---

## 왜 상담 내역을 보여주나요?

1. **후기 유도** — 상담 받은 카드마다 "후기 작성하기" 버튼이 붙어 후기 작성률을 높입니다.
2. **결제 투명성** — 언제·누구와·얼마나·몇 코인 썼는지 회원이 직접 확인할 수 있습니다.
3. **재상담 진입점** — 진행 중인 채팅방은 "채팅방 재입장하기"로 바로 돌아갈 수 있습니다.

---

## 화면 구성

### 세 가지 진입 경로

| 화면 | URL | 보이는 것 |
|------|-----|---------|
| 통합 상담내역 | `/mypage/history` | 전화+채팅 전부. 상단 탭(전체/전화상담/채팅상담)으로 필터 + 페이지네이션 |
| 전화상담 내역 | `/mypage/calls` | 전화상담만 (탭 없음, 최근 50건) |
| 채팅상담 내역 | `/mypage/chats` | 채팅상담만 (진행 중인 채팅방 포함) |

> 세 화면 모두 **같은 데이터(consultation 테이블)** 를 본다. 차이는 필터(`type`) 와 화면 구성뿐이다.

### 카드에 표시되는 항목

- **상담사** — 프로필 사진, 닉네임, 상담사 코드(csrid), 분야 뱃지(사주/타로/신점)
- **상담 시작 일시** — `2026.06.10 14:30` 형식
- **상담 종류** — 전화상담 / 채팅상담
- **사용 시간** — `00시간17분30초` 형식
- **사용 코인** — 해당 상담에 차감된 금액
- **후기 버튼** — 상태에 따라 "후기 작성하기" / "후기 보러가기" / "5분 이상 상담 후 후기 작성 가능"

---

## 후기 버튼이 바뀌는 규칙

| 상황 | 표시 |
|------|------|
| 후기 작성 완료 (review_id 있음) | "후기 보러가기" |
| 후기 미작성 + 사용 시간 5분(300초) 이상 | "후기 작성하기" |
| 후기 미작성 + 사용 시간 5분 미만 | "5분 이상 상담 후 후기 작성 가능" (버튼 없음, 안내 문구만) |
| 진행 중인 채팅방 (STAY/CNCH) | "채팅방 재입장하기" (후기 작성은 종료 후) |

> 5분 미만은 백엔드도 후기 작성을 막는다. 화면에서 미리 버튼을 숨겨 "후기 작성하기 눌렀더니 에러" UX를 차단한다.

---

## 빈 상태 처리

- 전화상담 없음 → "아직 전화상담 내역이 없습니다."
- 채팅상담 없음 → "아직 채팅상담 내역이 없습니다."
- 통합 내역 없음 → "상담 내역이 없습니다."

---

## 경우의 수별 설명

### Q. "통화는 했는데 상담 내역에 안 떠요"
→ 두 가지 경우입니다.
1. **연결 실패 통화는 일부러 숨깁니다.** 상담사 정보 없이 0초·0원으로 끝난 통화 로그는 회원 마이페이지에서 자동으로 빠집니다 (2026-05-29 정책). "0초 0원" 짜리 빈 카드가 보이지 않도록 한 처리입니다.
2. 정상 상담인데 안 보이면 사고 → 운영자 확인 필요.

### Q. "내가 상담사인데, 회원 상담내역에 내가 상담해준 건이 떠요"
→ 회원·상담사 두 역할을 같이 쓰는 분에게만 생기던 현상입니다. 현재는 **회원 시점으로 고정**되어 본인이 상담받은 건만 나옵니다. 두 역할을 쓰면 화면 상단에 핑크색 안내("회원 ⇄ 상담사 모드 전환 확인")가 한 줄 나옵니다. 일반 회원에겐 절대 안 보입니다.

### Q. "본인에게 상담 신청이 됐어요"
→ 과거에 회원=상담사 동일인이 자기 자신에게 상담을 거는 버그가 있었습니다 (2026-06-11 수정). 지금은 "본인에게 상담을 요청할 수 없습니다."로 차단됩니다.

### Q. "사용 시간/사용 코인이 0으로 떠요"
→ 종료 처리는 됐지만 실제 통화·채팅이 거의 없었던 경우입니다. 정상이면 0으로 표시될 수 있으나, 분 단위로 길게 쓴 상담이 0이면 사고 → 운영자 확인.

---

## 이건 정상인가요? 에러인가요?

| 상황 | 판단 |
|------|------|
| 종료된 상담만 보임 | ✅ 정상 (진행 전/연결 대기 건은 안 보임) |
| 연결 실패(상담사 없음·0초) 통화가 안 보임 | ✅ 정상 (2026-05-29 숨김 처리) |
| 진행 중 채팅방이 "재입장하기"로 보임 | ✅ 정상 |
| 5분 미만 상담에 후기 버튼 없음 | ✅ 정상 (5분 후기 정책) |
| 회원인데 내가 상담사로 표시 | ❌ 사고 — 역할 고정 깨짐 |
| 본인에게 상담 신청 가능 | ❌ 사고 — self 상담 차단 깨짐 |
