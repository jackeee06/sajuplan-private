# 추천인 시스템 v2 운영 가이드

## 한 줄로 답하면
**기존 상담사(A)가 신규 상담사(B)를 추천하면, B의 수익금 중 일부가 A에게 최대 6개월간 지급됩니다. 1~3개월 2%, 4~6개월 1%. 매월 관리자가 지급 버튼을 눌러야 실행됩니다.**

---

## 왜 추천인 시스템을 만들었나요?

1. **상담사 네트워크 확장** — 광고 없이 기존 상담사의 입소문으로 신규 상담사를 유입할 수 있습니다. 기존 상담사가 검증한 신규 상담사는 품질이 더 높을 가능성이 큽니다.
2. **상담사 이탈 방지** — 추천 관계가 살아있는 동안 추천자(A)가 활동해야 수당을 받으므로, A의 이탈 유인이 줄어듭니다.
3. **단계별 감소 구조** — 초기(1~3개월) 높은 수당으로 집중 지원 후, 이후(4~6개월) 낮은 수당으로 자립을 유도합니다. 6개월 후 완전 자립이 목표입니다.

---

## 추천 수당 정책 (현재 설정값)

| 기간 | 수당 | 계산 기준 |
|------|------|---------|
| 1~3개월 | B 수익금의 **2%** | 승인일 기준 월 카운트 |
| 4~6개월 | B 수익금의 **1%** | 승인일 기준 월 카운트 |
| 6개월 초과 | **없음** | 관계 자동 만료 |

> 예시: B가 이번 달 수익금 100만원 → A에게 2만원 지급 (1~3개월 기간 중)

---

## 추천 코드란?

- **형식**: 상담사 로그인 아이디(`mb_id`) 그대로 사용
- 별도 코드 발급 불필요 — 아이디가 곧 추천 코드
- **왜?** — 별도 코드 관리 시스템 없이 단순하게 운영하기 위해서입니다

신규 상담사 신청 시 "추천인 아이디" 입력란에 A의 mb_id를 입력하면 추천 관계 성립.

---

## 경우의 수별 설명

### Q. "추천 수당을 이번 달에 지급하려면?"
→ 매출현황 → 상담사 추천 수당 → **이번 달 지급대상** 탭 → 개별 또는 일괄 "지급" 버튼
→ 매월 **1~5일** 사이에 처리 권장 (정산 처리 시기와 맞춤)

### Q. "추천 관계가 몇 개월 됐는지 확인하려면?"
→ 추천 수당 페이지 → 활성 관계 탭 → `승인일` 컬럼 기준으로 계산

### Q. "A가 탈퇴하면 수당은?"
→ 추천 관계 비활성 처리 → 이후 수당 미지급

### Q. "B의 수익금이 0원이면?"
→ A에게도 0원 지급 (B 활동 없으면 수당 없음)

### Q. "추천 관계를 수동으로 등록하려면?"
→ 추천 수당 페이지 → 새 추천 관계 등록 버튼 → 추천자(A)·피추천자(B) 선택

### Q. "6개월이 지나 연장하고 싶으면?"
→ 현재 연장 기능 미구현. 6개월 후 자동 만료. (추후 검토 예정)

---

## 지급 흐름

```
매월 1~5일
  ↓
관리자 → 추천 수당 페이지 → 이번 달 지급대상 탭
  ↓
지급액 미리보기 확인
  ↓
지급 버튼 클릭
  ↓
A의 수익금(earning_balance)에 추가
  ↓
다음 정산(매월 1일)에 일반 정산과 함께 원천세 차감 후 입금
```

> ⚠️ 추천 수당은 **수익금(earning_balance)** 으로 지급됩니다.
> 일반 정산과 동일하게 원천세(3.3%) 차감 후 입금됩니다.

---

## 정산서에서 추천 수당 확인

정산 이력 페이지에서 A의 해당 월 정산 내역에 "추천수당" 항목으로 별도 표시됩니다.

---

## 이건 정상인가요? 에러인가요?

| 상황 | 판단 |
|------|------|
| 1~3개월에 2% 지급 | ✅ 정상 |
| 4~6개월에 1% 지급 | ✅ 정상 |
| 7개월째 지급대상에 나타남 | ❌ 정책 오류 — 즉시 확인 |
| 미지급 건이 다음 달로 이월 | ⚠️ 수동 처리 필요 (자동 이월 없음) |
| B가 활동 안 해서 수당이 0원 | ✅ 정상 |

---

## 운영자가 확인하는 위치

| 작업 | 메뉴 위치 |
|------|---------|
| 전체 추천 관계 목록 | 매출현황 → 상담사 추천 수당 |
| 이번 달 지급 처리 | 매출현황 → 상담사 추천 수당 → 이번 달 지급대상 탭 |
| 정책 요율·기간 변경 | 추천 수당 → 정책 설정 (슈퍼관리자 전용) |
