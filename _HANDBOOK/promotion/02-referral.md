# 추천인 시스템

## 한 줄로 답하면
**기존 회원이 신규 회원을 추천하면 양쪽에 무료 코인 보상. 가입 시 추천인 코드 입력으로 매칭.**

## 동작

```
기존 회원 (마이페이지에서 본인 추천인 코드 확인)
  ↓
지인에게 추천인 코드 공유
  ↓
신규 회원이 가입 시 추천인 코드 입력
  ↓
사주플랜:
  1. 신규 회원 등록 + 추천인 매칭
  2. 신규에게 보상 (무료 코인)
  3. 추천인에게 보상 (무료 코인)
```

## 이런 상황을 보셨나요?

### "추천인 코드 입력했는데 보상 안 들어왔어요" 라고 신규 회원 클레임

→ 가능 원인:
1. 코드 잘못 입력 (오타)
2. 시스템 사고

→ 답변: "추천인 코드 확인하겠습니다." 그 후 사이드바 → 매출현황 → 프로모션 에서 추천 매칭 검토. 필요 시 수동 보상 적립.

### "내 추천인 코드 어디서 봐요?"

→ 회원 마이페이지 → 추천하기 (또는 설정) 에서 본인 코드 확인.

### "같은 사람이 여러 번 추천 보상 받아요"

→ 어뷰징 의심. 사이드바 → 매출현황 → 프로모션 에서 이상 패턴 검토.

## 이건 정상인가요? 에러인가요?

| 상황 | 답변 |
|---|---|
| 추천인 코드 입력 가입 → 양쪽 보상 | ✅ 정상 |
| 코드 잘못 입력 → 무보상 가입 | ✅ 정상 (코드 매칭 실패) |
| 자기 자신 코드 입력 시도 | ❌ 차단 |
| 같은 사람 여러 번 추천 보상 | ⚠️ 어뷰징 의심. 점검 |
| 추천 보상이 paid_balance 로 적립 | ❌ 정책 위반 (free 로 가야 함) |

## 운영자가 확인하는 위치

| 확인 항목 | 메뉴 위치 |
|---|---|
| 추천 통계 | 사이드바 → 매출현황 → 프로모션 |
| 추천인 보상 정책 설정 | 사이드바 → 매출현황 → 프로모션 → 설정 |
| 어뷰징 의심 케이스 | 같은 phone / 같은 IP 다수 추천 |
