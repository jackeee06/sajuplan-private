# 쿠폰 시스템

## 한 줄로 답하면
**관리자가 쿠폰존(캠페인)을 만들면 → 회원에게 자동 또는 코드로 발급 → 회원이 마이페이지에서 "사용" 버튼을 눌러야 코인으로 전환됩니다.**

---

## 1. 쿠폰 종류 (2가지)

| 종류 | 동작 방식 | 코드 필요? | 주요 용도 |
|---|---|---|---|
| **다운로드 쿠폰** | 관리자가 특정 회원에게 직접 발급 | ❌ 불필요 | 회원가입 축하, 이벤트 당첨 |
| **코드입력 쿠폰** | 회원이 마이페이지에서 코드 직접 입력 | ✅ 필수 | SNS 이벤트, 제휴사, 프로모션 배포 |

> **쿠폰 코드 형식**: `XXXX-XXXX-XXXX-XXXX` (자동 생성, 대문자+숫자, I·O·0 제외)

---

## 2. 회원가입 쿠폰 (자동 발급)

- 신규 회원 가입 완료 시 **코드 없이 자동 지급** — 관리자 조작 불필요
- **현재 설정**: 5,000 코인 / 유효기간 **7일**
- 가입 완료 직후 알림톡(`coupon_signup_v1`)으로 자동 발송 (BizM 승인 후 작동)
- 중복 발급 방지: 동일 회원에게 2번 발급되지 않음
- 설정 변경: 관리자 → 쿠폰존 관리 → "회원가입 쿠폰" 클릭 → 포인트/유효기간 수정

---

## 3. 쿠폰 → 코인 전환 흐름

```
1. 회원가입 완료 (또는 코드 입력)
   ↓
2. 쿠폰이 회원 마이페이지에 표시됨
   ↓
3. 회원이 "사용" 버튼 클릭
   ↓
4. 코인 즉시 충전 (잔액에 추가)
```

> ⚠️ **중요**: 쿠폰을 받았다고 자동으로 코인이 충전되지 않습니다. 회원이 반드시 "사용" 버튼을 눌러야 합니다. 유효기간 내 사용하지 않으면 자동 소멸됩니다.

---

## 4. 쿠폰존(캠페인) 만들기 — 순서

1. 관리자 → **쿠폰존 관리** → **+ 새 쿠폰존 추가**
2. 입력 항목:
   - **쿠폰 이름**: 회원에게 보이는 이름 (예: "여름 이벤트 쿠폰")
   - **지급 코인 수**: 회원이 "사용" 시 충전될 코인 액수
   - **쿠폰 종류**: 다운로드 또는 코드입력
   - **유효기간(일수)**: 발급일로부터 N일 (예: 7)
   - **사용 기간**: 쿠폰존 자체가 유효한 기간 (이 기간 이후 코드 입력 불가)
   - **발급 대상 회원**: 다운로드 쿠폰 선택 시 회원 직접 지정
3. 저장 → 코드입력 쿠폰이면 코드 자동 생성됨 → SNS 등에 배포

> ⚠️ **주의**: 쿠폰존 유형(다운로드/코드입력)은 저장 후 변경 불가. 신중히 선택.

> ⚠️ **주의**: 쿠폰존 삭제 시 해당 캠페인으로 발급된 쿠폰 이력도 삭제됩니다. 함부로 삭제 금지.

---

## 5. 알림톡 현황

| 템플릿 코드 | 용도 | 상태 |
|---|---|---|
| `coupon_signup_v1` | 회원가입 쿠폰 발급 알림 | 🟡 BizM 검수 진행 중 (승인 후 자동 발송) |
| `coupon_req_v2` | 일반 쿠폰 발급 알림 | 🔴 BizM 반려 (추후 재신청 예정) |

> ⚠️ 알림톡 승인 전이면 쿠폰을 발급해도 회원이 직접 마이페이지를 방문해야 확인 가능합니다.
> BizM 포털([bizmsg.kr](https://bizmsg.kr))에서 검수 상태를 주기적으로 확인하세요.

---

## 6. 발급/사용 현황 확인 방법

| 확인 항목 | 위치 |
|---|---|
| 전체 쿠폰존 목록 | 관리자 → 쿠폰존 관리 |
| 쿠폰존별 발급 수 | 쿠폰존 목록 "다운로드" 컬럼 |
| 쿠폰존 발급 대상 회원 목록 | 쿠폰존 클릭 → 발급 회원 탭 |
| 회원 개인 쿠폰 현황 | 고객 리스트 → 회원 클릭 → 쿠폰 탭 |
| 전체 쿠폰 발급/사용 이력 | 관리자 → 쿠폰 목록 (회원 ID / 코드 / 이름으로 검색) |

## 7. 자주 묻는 질문 (FAQ)

### "회원이 쿠폰 받았는데 코인이 안 충전됐어요"
→ 쿠폰은 받은 즉시 코인이 되지 않습니다. 회원이 마이페이지 → 쿠폰 → "사용" 버튼을 눌러야 충전됩니다. 버튼을 눌렀는지 확인하세요.

### "유효기간이 지난 쿠폰인데 사용하게 해달래요"
→ 유효기간 만료 쿠폰은 시스템에서 자동 차단됩니다. 재발급을 원하면 새 쿠폰존을 만들어 발급해 주세요.

### "회원이 쿠폰 코드를 입력했는데 오류가 나요"
→ 가능 원인: 코드 오타 / 유효기간(사용기간) 만료 / 이미 사용한 코드 / 같은 쿠폰존에서 이미 발급받은 경우. 쿠폰존 관리에서 해당 코드의 상태를 확인하세요.

### "회원이 쿠폰으로 충전 후 환불 요청을 해요"
→ 현재 쿠폰으로 충전된 코인의 환불 정책은 미명시 상태입니다. 운영팀에서 결정 후 이 문서를 업데이트하세요.

### "가입 쿠폰을 5,000 코인에서 다른 금액으로 바꾸고 싶어요"
→ 쿠폰존 관리 → "회원가입 쿠폰" 클릭 → 포인트 추가금액 수정 → 저장. 이후 가입하는 회원부터 적용됩니다.
