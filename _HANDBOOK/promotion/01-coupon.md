# 쿠폰 시스템

## 한 줄로 답하면
**운영자가 쿠폰을 발급 → 회원이 보유 → 충전 시 사용. 쿠폰존(그룹) 단위로 관리. 회원이 쿠폰 코드 입력해서 받기도 가능.**

## 쿠폰 종류

| 발급 방법 | 예시 |
|---|---|
| 자동 발급 | 가입 환영, 출석 보너스, 이벤트 |
| 수동 발급 | 운영자가 특정 회원 / 그룹에게 |
| 쿠폰존 등록 | 회원이 코드 입력해서 받기 |

## 이런 상황을 보셨나요?

### "회원에게 쿠폰 보내고 싶어요"

→ 사이드바 → 쿠폰존 관리 → 쿠폰존 선택 → 회원 선택 → 발급.

### "쿠폰 사용했는데 안 적용됐어요" 라고 회원 클레임

→ 가능 원인: 유효기간 만료 / 이미 사용 / 사용 조건 미충족.

→ 답변: "쿠폰 정보 확인하겠습니다. 사이드바 → 회원현황 → 고객 리스트 → 상세 → 쿠폰 탭 에서 회원 보유 쿠폰 조회."

### "쿠폰 발급 알림 보내고 싶어요"

→ 메모리상 `coupon_req_v2` 사용 안 하기로 결정. 인앱 알림으로 안내.

## 이건 정상인가요? 에러인가요?

| 상황 | 답변 |
|---|---|
| 회원 쿠폰 발급 + 알림 | ✅ 정상 |
| 유효기간 지난 쿠폰 사용 시도 → 거부 | ✅ 정상 |
| 같은 쿠폰 두 번 사용 시도 → 거부 | ✅ 정상 (used_at 마킹) |
| 쿠폰 사용 후 환불 시 쿠폰 복원? | 정책 미명시. 운영 시작 후 결정 |

## 운영자가 확인하는 위치

| 확인 항목 | 메뉴 위치 |
|---|---|
| 쿠폰존 관리 | 사이드바 → 쿠폰존 관리 |
| 회원 보유 쿠폰 | 사이드바 → 회원현황 → 고객 리스트 → 상세 |
| 쿠폰 사용 통계 | 사이드바 → 쿠폰존 관리 → 통계 |
