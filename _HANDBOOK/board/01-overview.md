# 게시판 시스템 (후기·문의·공지·이벤트·FAQ)

## 한 줄로 답하면
**사주플랜은 5종 게시판 통합 운영. 회원이 후기·문의 작성, 운영자가 공지·이벤트·FAQ 관리. 신고는 별도 처리.**

## 후기 코인 정책 (현재값)

| 항목 | 값 |
|---|---|
| 일반 후기 작성 | 500코인 적립 |
| 사진 첨부 시 | +500코인 (합계 1,000코인) |
| 작성 조건 | 5분 이상 진행한 상담만 / 같은 상담 1건만 (중복 차단) |
| 작성 기한 | 상담 종료 후 7일 이내 |
| 삭제 시 | 작성 5분 이내 + 상담사 답변 전이면 삭제 가능 → 지급 코인 회수 |
| 베스트 후기 (관리자 선정) | 작성자에게 10,000코인 추가 지급 (1회) |

## 2026-06-10 수정 (후기 버그 3건 — 운영자용 요약)

### 1. 후기 코인 이중 지급 차단
- 증상: 같은 상담에 후기를 여러 번 쓰면 매번 500코인(사진이면 1,000코인)이 또 지급됐음 (중복 차단이 안 됨).
- 원인: 후기 데이터의 상담번호가 잘못된 형식으로 저장돼, 중복 검사가 항상 빈손으로 통과됐음.
- 조치: 저장 방식 정상화 + 중복 검사 강화 + 기존 잘못 저장된 6건 정정. 이제 같은 상담 후기는 1건만 인정.

### 2. 후기 사진 업로드 가능해짐 (지금까지 한 번도 안 됐음)
- 증상: 모바일 카메라 사진(3~8MB)을 올리면 "용량 초과" 또는 "업로드 실패" — 사진 후기가 사실상 불가능했음.
- 원인: 서버가 받는 사진 용량 한도가 5MB로 너무 작아, 요즘 폰 사진은 전부 거부됐음.
- 조치: 한도를 5MB → 30MB로 확대. 동시에 사진을 자동으로 가볍게 변환(webp)해서, 8.7MB 원본도 0.33MB로 줄여 저장·표시. 21MB 사진 업로드까지 성공 확인.

### 3. 듀얼역할자(상담사가 회원으로 후기 작성) "상담사가 아닙니다" 오류
- 증상: 상담사 본인이 회원으로 다른 상담사에게 상담받고 마이페이지에서 후기를 쓰면 엉뚱한 에러가 났음.
- 원인: 회원 상담내역 조회 시 "회원 시점"임을 명시하지 않아, 듀얼역할자는 "상담사 시점"으로 잘못 조회됐음.
- 조치: 회원 상담내역은 항상 회원 시점으로 조회 + 후기 작성 검증을 실제 상담 기록 기준으로 교정. 정상 작성됨.

## 신고 처리

회원이 후기 / 문의 글 / 댓글을 신고하면:
1. 신고 카운트 증가
2. 임계값 (예: 5건) 도달 시 자동 숨김
3. 운영자가 사이드바 → 상담관리 → 후기 신고 관리 또는 게시판 신고 관리 에서 검토
4. 복원 / 영구삭제 / 사용자 제재 결정

## 이런 상황을 보셨나요?

### "회원이 부적절한 후기 썼대요" 라고 다른 회원이 신고

→ 신고 누적되면 자동 숨김. 운영자가 사이드바 → 상담관리 → 후기 신고 관리 에서 검토 후 처리.

### "공지사항 안 보인다고 회원이 클레임"

→ 가능 원인: 공지 미공개 상태. 사이드바 → 게시판관리 → 공지사항 관리 에서 발행 상태 확인.

### "FAQ 자주 묻는 질문 추가하고 싶어요"

→ 사이드바 → 게시판관리 → FAQ 관리 → 새 글 등록.
