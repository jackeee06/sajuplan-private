# 게시판 시스템 (후기·문의·공지·이벤트·FAQ)

## 한 줄로 답하면
**사주플랜은 5종 게시판 통합 운영. 회원이 후기·문의 작성, 운영자가 공지·이벤트·FAQ 관리. 신고는 별도 처리.**

## 게시판 5종

| 종류 | 작성자 | 운영자 메뉴 |
|---|---|---|
| 상담 후기 | 회원 | 사이드바 → 상담관리 → 상담후기 관리 |
| 회원 1:1 문의 | 회원 | 사이드바 → 상담관리 → 상담문의 |
| 상담사 1:1 문의 | 상담사 | 사이드바 → 상담관리 → 1:1문의(상담사) |
| 공지사항 | 운영자 | 사이드바 → 게시판관리 → 공지사항 관리 |
| 이벤트 | 운영자 | 사이드바 → 게시판관리 → 이벤트 관리 |
| FAQ | 운영자 | 사이드바 → 게시판관리 → FAQ 관리 |

## 신고 처리

회원이 후기 / 문의 글 / 댓글을 신고하면:
1. 신고 카운트 증가
2. 임계값 (예: 5건) 도달 시 자동 숨김
3. 운영자가 사이드바 → 상담관리 → 후기 신고 관리 또는 게시판 신고 관리 에서 검토
4. 복원 / 영구삭제 / 사용자 제재 결정

## 이런 상황을 보셨나요?

### "회원이 부적절한 후기 썼대요" 라고 다른 회원이 신고

→ 신고 누적되면 자동 숨김. 운영자가 사이드바 → 상담관리 → 후기 신고 관리 에서 검토 후 처리.

### "공지사항 안 보인다고 회원이 클레임"

→ 가능 원인: 공지 미공개 상태. 사이드바 → 게시판관리 → 공지사항 관리 에서 발행 상태 확인.

### "FAQ 자주 묻는 질문 추가하고 싶어요"

→ 사이드바 → 게시판관리 → FAQ 관리 → 새 글 등록.

## 이건 정상인가요? 에러인가요?

| 상황 | 답변 |
|---|---|
| 회원 후기 작성 → 상담사에게 알림톡 | ✅ 정상 |
| 신고 5건 누적 → 자동 숨김 | ✅ 정상 |
| 운영자가 글 영구삭제 → 회원 알림 | 정책 결정. 현재 미구현 |
| 공지 발행 → 회원 화면 즉시 노출 | ✅ 정상 |

## 운영자가 확인하는 위치

| 확인 항목 | 메뉴 위치 |
|---|---|
| 상담 후기 관리 | 사이드바 → 상담관리 → 상담후기 관리 |
| 후기 신고 | 사이드바 → 상담관리 → 후기 신고 관리 |
| 회원 문의 | 사이드바 → 상담관리 → 상담문의 |
| 상담사 문의 | 사이드바 → 상담관리 → 1:1문의(상담사) |
| 공지 / 이벤트 / FAQ | 사이드바 → 게시판관리 → 각각 |
| 게시판 신고 | 사이드바 → 게시판관리 → 게시판 신고 관리 |
| 글·댓글 현황 (통합) | 사이드바 → 게시판관리 → 글·댓글 현황 |
