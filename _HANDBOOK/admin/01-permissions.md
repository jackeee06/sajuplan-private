# 관리자 권한 (슈퍼 / 일반)

## 한 줄로 답하면
**슈퍼관리자 (사장님 본인) + 일반관리자 (운영팀). 슈퍼 전용은 ① 슈퍼 승격 ② 영업이익 시뮬레이터 만. 일상 돈업무는 일반관리자도 OK.**

## 권한 차이

### 슈퍼관리자 전용
1. 다른 관리자를 슈퍼로 승격
2. 영업이익 시뮬레이터 (비밀 단가·마진 영역)

### 일반관리자 전권
- 회원 관리 (정지, 정보 수정)
- 결제, 환불, 정산 처리
- 콘텐츠 (공지, 이벤트, FAQ)
- 알림 발송
- 상담사 신청 / 승인
- 게시판 신고 처리

## 이런 상황을 보셨나요?

### "일반관리자에게 매출 보여줄까요?"

→ 일반관리자도 매출 / 환불 / 정산 등 일반 운영 정보 OK. 다만 영업이익률 / 비밀 단가 / 회사 마진은 슈퍼만.

### "새 관리자 추가하고 싶어요"

→ 슈퍼관리자 (사장님) 만 가능. 사이드바 → 권한관리 → 관리자 계정 → 등록 → 일반관리자로 등록.

### "운영팀이 영업이익 시뮬레이터 보려고 해요"

→ 차단됨 (정상). 사장님이 결정한 정보 비밀이라 슈퍼만 접근.

## 이건 정상인가요? 에러인가요?

| 상황 | 답변 |
|---|---|
| 일반관리자가 환불 처리 | ✅ 정상 |
| 일반관리자가 매출 통계 조회 | ✅ 정상 |
| 일반관리자가 영업이익 시뮬레이터 접근 시도 → 차단 | ✅ 정상 |
| 슈퍼관리자가 사라짐 (사장님 본인) | ❌ 시스템 사고. DB 직접 조회 + 슈퍼 권한 복원 |
| 일반관리자가 다른 관리자 추가 시도 → 차단 | ✅ 정상 |

## 운영자가 확인하는 위치

| 확인 항목 | 메뉴 위치 |
|---|---|
| 관리자 계정 목록 (슈퍼 전용 페이지) | 사이드바 → 권한관리 → 관리자 계정 |
| 본인 권한 확인 | 사이드바 상단 또는 마이페이지 |
