# 영업이익 시뮬레이터 (슈퍼관리자 전용)

## 한 줄로 답하면
**가정 기반으로 미래 매출·영업이익을 미리 계산해보는 도구. 사장님 의사결정용. 일반관리자는 접근 불가.**

## 입력 변수

- 회원 수 (MAU)
- 회원당 평균 결제
- 채팅 / 통화 비율
- 상담사 단가 (등급별)
- 마진 비율
- 환불율
- 운영비 (인건비, 외부 서비스)

## 계산

```
매출 = 회원 수 × 회원당 결제 × (1 - 환불율)
정산 비용 = 매출 × 채팅·통화 비율 × (1 - 마진)
영업이익 = 매출 - 정산 비용 - 운영비
영업이익률 = 영업이익 / 매출
```

## 활용 시점

### 1. 마케팅 예산 결정
- "회원 1만명 늘리려면 광고비 X 원" — 그 결과 영업이익 시뮬

### 2. 단가 / 마진 조정
- 상담사 단가 올리면 영업이익 어떻게 변하나
- 마진 비율 조정 시 손익분기점

### 3. 손익분기점 (BEP)
- 회원 수가 몇 명 되어야 흑자?

### 4. 외부 투자 자료
- 투자자에게 사업 전망 제시 (비밀 수치는 가림)

## 현재 단계

테스트 단계 → 실측 데이터 부족. 시뮬레이터가 **미래 가정 도구로 더 유의미**.

## 이건 정상인가요? 에러인가요?

| 상황 | 답변 |
|---|---|
| 슈퍼관리자 접근 → 시뮬레이션 가능 | ✅ 정상 |
| 일반관리자 접근 시도 → 차단 (메뉴 자체 안 보임) | ✅ 정상 |
| 시뮬레이션 결과 = 실측 매출과 동일 | ⚠️ 우연일 가능성. 시뮬레이터는 가정 기반 |
