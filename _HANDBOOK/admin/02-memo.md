# 관리자 메모장

## 한 줄로 답하면
**각 관리자별 개인 메모장. 본인만 보임. 이미지 인라인 업로드 지원. 운영 임시 메모용.**

## 동작

- 사이드바 → 메모장 진입
- Toast UI Editor (워드 같은 편집기)
- 내용 작성 → 저장 (또는 자동 저장)
- 이미지 드래그&드롭 또는 붙여넣기 → 자동 업로드

## 이런 상황을 보셨나요?

### "회원 클레임 처리 메모하고 싶어요"

→ 메모장에 자유롭게 작성. 본인만 봄. 단, 영구 기록 필요한 정책은 사장님께 보고 후 운영 바이블에 박제.

### "다른 관리자와 메모 공유하고 싶어요"

→ 현재는 개인 메모만. 공유 메모 기능 없음 (향후 검토).

### "운영 바이블이랑 메모장 차이가 뭐예요?"

| 메모장 | 운영 바이블 |
|---|---|
| 개인 임시 | 영구 정책 |
| 본인만 봄 | 모든 관리자 |
| 자유 형식 | 4섹션 일관 포맷 |
| 운영 중 빠른 기록 | 운영 중 빠른 참조 |

## 이건 정상인가요? 에러인가요?

| 상황 | 답변 |
|---|---|
| 본인 메모 정상 로딩 | ✅ 정상 |
| 이미지 업로드 (10MB 이하) | ✅ 정상 |
| 다른 관리자 메모 보임 | ❌ **사고**. 권한 가드 점검 |
| 10MB 초과 이미지 거부 | ✅ 정상 |

## 운영자가 확인하는 위치

| 확인 항목 | 메뉴 위치 |
|---|---|
| 본인 메모장 | 사이드바 → 메모장 |
