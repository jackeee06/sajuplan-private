# 환경설정 / 약관·처리방침

## 한 줄로 답하면
**환경설정 = 외부 서비스 키 / 시스템 정책 값. 내용 관리 = 약관·개인정보처리방침 (법적 문서).**

## 환경설정

운영 중 수정 가능한 값:
- BizM 발신 프로필
- 알림 정책 임계값
- 채팅 시간 옵션 토글 (15·30·45·60분)
- 출석 보상 코인 양
- G 정책 한도 / 임계값

→ 정확한 항목은 사이드바 → 환경설정 → 기본환경설정 직접 확인.

## 내용 관리 (약관)

- 이용약관
- 개인정보처리방침
- 환불 정책 안내
- 마케팅 동의 안내

→ 회원 가입 시 동의 + 사용자 화면 footer 링크.

## 이런 상황을 보셨나요?

### "약관 수정하고 싶어요"

→ 사이드바 → 환경설정 → 내용 관리 → 항목 선택 → 수정 → 저장. 사용자 화면 즉시 반영.

### "큰 약관 변경인데 회원 동의 다시 받아야 하나요?"

→ 법적 검토 필요. 큰 변경 시 사전 공지 + 재동의 강제 권장. 현재 자동 재동의 강제 X.

### "환경설정 값 잘못 수정했어요"

→ 운영자 수동 복원. 사고 매뉴얼 (`system/03-incident`) 참조. DB 백업 활용.

## 이건 정상인가요? 에러인가요?

| 상황 | 답변 |
|---|---|
| 약관 수정 → 사용자 화면 즉시 반영 | ✅ 정상 |
| 환경설정 수정 → 시스템 즉시 적용 | ✅ 정상 (대부분) |
| 큰 약관 변경 후 회원 재동의 없이 운영 | ⚠️ 법적 위험 가능 |
| 환경설정 실수로 외부 서비스 키 삭제 | ❌ 사고. 즉시 복원 |

## 운영자가 확인하는 위치

| 확인 항목 | 메뉴 위치 |
|---|---|
| 기본환경설정 | 사이드바 → 환경설정 → 기본환경설정 |
| 약관 / 처리방침 관리 | 사이드바 → 환경설정 → 내용 관리 |
