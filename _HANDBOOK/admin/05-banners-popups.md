# 배너·팝업 관리

## 한 줄로 답하면
**회원 화면 상단 배너 (홈) + 진입 시 팝업 레이어. 운영자가 등록·기간·노출 위치 설정.**

## 종류

### 홈 배너
- Home 페이지 상단 슬라이드
- 마케팅·이벤트 안내용
- 자동 슬라이드 (3-5초)

### 팝업 레이어
- 사주플랜 진입 시 자동 노출
- "오늘 하루 보지 않기" / "닫기" 옵션
- localStorage 기반 노출 제한

### 시스템 배너 (별도)
- 점검 안내 (노랑)
- 단골 상담사 접속 (핑크)
- 상담사 incoming 채팅 (핑크)
- → 운영자가 직접 만들지 않고 시스템 상태 따라 자동

## 이런 상황을 보셨나요?

### "이벤트 배너 새로 올리고 싶어요"

→ 사이드바 → 기타 → 배너관리 → 새 배너 등록. 이미지·기간·링크 입력. 활성 토글로 즉시 노출.

### "팝업 너무 자주 떠요"

→ 팝업 노출 정책 (예: 1일 1회 / 7일 1회) 설정. 사이드바 → 기타 → 팝업레이어 관리.

### "배너 잘못 올라갔어요 (실수)"

→ 즉시 사이드바 → 기타 → 배너관리 → 비활성 토글. 캐시 갱신은 사용자별 시간 차이 있음.

## 이건 정상인가요? 에러인가요?

| 상황 | 답변 |
|---|---|
| 등록 → 즉시 회원 화면 노출 | ✅ 정상 |
| 기간 종료 → 자동 숨김 | ✅ 정상 |
| 사용자가 팝업 닫기 → localStorage 저장 | ✅ 정상 |
| 비활성 토글했는데 일부 사용자에게 보임 | ⚠️ 브라우저 캐시. 시간 지나면 갱신 |
| 외부 URL 클릭이 안 됨 (WebView 한계) | ⚠️ openExternalUrl() 호출 필요 |

## 운영자가 확인하는 위치

| 확인 항목 | 메뉴 위치 |
|---|---|
| 배너 등록·관리 | 사이드바 → 기타 → 배너관리 |
| 팝업 레이어 관리 | 사이드바 → 기타 → 팝업레이어 관리 |
| 사주메인 (홈 구성) | 사이드바 → 기타 → 사주메인관리 |
