# 운영 바이블 AI 는 어떻게 동작해요?

## 한 줄로 답하면
**운영 바이블 (46개 항목) + 실제 사주플랜 DB 데이터를 기반으로 자연어 질문에 답하는 AI 도구. 사이드바 → 🤖 운영 바이블 AI 에서 사용.**

## 무엇이 가능해요?

### 1. 정책 질문 (바이블만으로 답)
- "회원이 5초 안에 채팅 끝났는데 환불해야 해요?"
- "선지급 수수료 몇 %예요?"
- "상담사 등급 어떻게 올라가요?"

### 2. 데이터 진단 질문 (DB 자동 조회)
- "01012345678 회원 통화 시간이랑 차감 이상하대요"
- "어제 매출 얼마예요?"
- "특정 상담사 한 달 정산 얼마예요?"

### 3. 복합 진단 질문
- "회원이 결제했는데 코인 안 들어왔다고 해요" → 결제 이력 + 코인 변동 이력 자동 조회 + 정책 비교 + 진단

## 사용 방법

### 1단계 — 사이드바 → 🤖 운영 바이블 AI
### 2단계 — 자연어로 질문 입력
### 3단계 — AI 답변 + 출처 + (필요시) DB 조회 결과 확인

답변에 포함되는 정보:
- **답변 본문**
- **📚 출처**: 참조한 운영 바이블 항목 (클릭하면 해당 항목 이동)
- **🔍 실행된 DB 조회**: 자동 실행된 SQL 펼쳐서 확인 가능 (감사 추적)

## 이런 상황을 보셨나요?

### "AI 답변이 너무 짧아요"

→ 환경설정에서 max_tokens 늘리기. 슈퍼관리자가 사이드바 → AI 설정 (슈퍼) 에서 조정.

### "API 키 만료됐어요" 같은 에러

→ 슈퍼관리자가 사이드바 → AI 설정 (슈퍼) 에서 새 키 입력.

### "비활성 상태입니다" 안내

→ 슈퍼관리자가 AI 설정 페이지에서 활성화 토글 ✅ 필요.

### "옛 대화 다시 보고 싶어요"

→ 좌측 사이드 패널에 본인이 했던 옛 대화 목록. 클릭 = 이어보기 가능.

## 이건 정상인가요? 에러인가요?

| 상황 | 답변 |
|---|---|
| AI 답변에 출처 표시 안 됨 | ❌ **사고**. 출처 강제 정책 위반 |
| 출처가 클릭이 안 됨 | ⚠️ 일부 슬러그 깨짐. 새로고침 또는 보고 |
| "DB 조회 실패" 메시지 | ⚠️ 정상 동작 (SELECT 외 차단됨). AI 가 다시 시도하면 됨 |
| 답변 시간 5초 이상 | ✅ 정상 (DB 조회 + LLM 처리 시간) |
| API 한도 초과 알림 | ⚠️ Anthropic 측 크레딧 소진. 슈퍼관리자 충전 필요 |

## 안전 가드

운영 바이블 AI 는 **읽기 전용**입니다:

| 안전 가드 | 동작 |
|---|---|
| 데이터 변경 차단 | INSERT/UPDATE/DELETE 모두 차단. SELECT 만 |
| 민감 정보 마스킹 | phone 뒷자리, 비밀번호, 카드번호 자동 ✱ 처리 |
| 조회 제한 | 한 번에 최대 100행만 |
| 응답 시간 제한 | DB 조회 10초 초과 시 자동 중단 |
| 출처 강제 | 모든 답변에 참조한 바이블 항목 표시 |
| 컨텍스트 외 답변 거부 | 바이블에 없는 정보는 "모릅니다" 답변 |

→ AI 가 사주플랜 데이터를 **읽기만** 함. 실수로 데이터 망가뜨릴 일 없음.

## 비용

| 모델 | 1 query 비용 | 월 추정 (운영자 일 10-20 query) |
|---|---|---|
| Claude Sonnet 4.6 (권장) | 약 0.003 ~ 0.01 달러 | 약 1,500 ~ 5,000원 |
| Claude Haiku 4.5 (저렴) | 약 0.001 ~ 0.003 달러 | 약 500 ~ 1,500원 |

→ 일반 운영자 1명 일 사용량 기준 **월 1-5천원** 수준. Anthropic 콘솔에서 월 cap 설정 가능 (예: 월 $1 한도).

## 운영자가 확인하는 위치

| 확인 항목 | 메뉴 위치 |
|---|---|
| 자연어 질문 + 답변 | 사이드바 → 🤖 운영 바이블 AI |
| API 키 설정 (슈퍼만) | 사이드바 → AI 설정 (슈퍼) |
| 옛 대화 다시 보기 | 운영 바이블 AI 페이지 좌측 사이드 패널 |
| 모델 / max_tokens 조정 (슈퍼만) | AI 설정 페이지 |

## 자주 묻는 질문 (FAQ)

### Q. "AI 가 잘못 답하면 어쩌나요?"

A. 모든 답변에 출처 표시. 운영자가 출처 항목 클릭해서 원본 정책 직접 확인 가능. 답변에 의문 들면 항상 출처로 검증.

### Q. "회원 데이터를 외부로 보내는 거 아닌가요?"

A. 질문 + 관련 바이블 + 필요한 DB 조회 결과만 Anthropic 측 (Claude) 에 전송. 민감 정보 (phone 뒷자리, 비밀번호 등) 는 자동 마스킹 후 전송. Anthropic 은 데이터 학습 X (API 정책).

### Q. "Anthropic 결제 안 되면?"

A. 평가 액세스 (Free) 또는 OpenAI 등 다른 LLM 으로 전환 가능 (별도 작업). 또는 운영 바이블 (`📖`) 페이지에서 키워드 검색으로 직접 찾기.

### Q. "다른 운영자와 대화 공유 가능?"

A. 현재 미구현. 본인 대화만 본인이 봄. 향후 공유 기능 검토 가능.
