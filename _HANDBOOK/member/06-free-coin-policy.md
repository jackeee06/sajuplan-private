# 회원이 무료로 코인을 받을 수 있는 방법은 무엇인가요?

## 한 줄로 답하면
**결제 없이도 출석·후기·쿠폰·관리자 지급 등 10가지 경로로 무료 코인을 받을 수 있습니다. 모두 free_balance(무료 코인)로 적립됩니다.**

---

## 각 경로 상세 설명

### 1. 출석 보상

회원이 사주플랜 앱에 접속해 출석 버튼을 누르면 하루 1회 코인 적립.
가입 후 **3일 이상 경과**해야 출석 가능.

#### 회원 출석 보상표

| 상황 | 지급 코인 | 비고 |
|------|---------|------|
| 매일 출석 | **+100 코인** | 기본 지급 |
| 연속 5일 달성 | **+200 코인 추가** | 5일째 당일에 기본 100 + 보너스 200 = 300 코인 |
| 연속 10일 달성 | **+300 코인 추가** | 10일째 당일 400 코인 |
| 연속 15일 달성 | **+400 코인 추가** | 15일째 당일 500 코인 |
| 연속 20일 달성 | **+500 코인 추가** | 20일째 당일 600 코인 |
| 30일 달성 | **10,000 코인 쿠폰** | 쿠폰으로 발급, 14일 이내 직접 사용해야 코인 전환 |

#### 상담사 출석 보상표

| 상황 | 지급 코인 | 비고 |
|------|---------|------|
| 매일 출석 | **+50 코인** | 회원의 절반 |
| 연속 5일 달성 | **+100 코인 추가** | |
| 연속 10일 달성 | **+150 코인 추가** | |
| 연속 15일 달성 | **+200 코인 추가** | |
| 연속 20일 달성 | **+250 코인 추가** | |
| 30일 달성 | 없음 | 상담사는 30일 쿠폰 미지급 |

#### 주의사항
- 같은 날 2번 출석 불가 (자정 KST 기준 리셋)
- 출석을 건너뛰면 연속일 초기화 (5일 하다 1일 쉬면 0부터 재시작)
- 관리자 설정: **회원현황 → 출석 관리 → 정책 설정**

---

### 2. 쿠폰

쿠폰을 받아도 **마이페이지 → 내 쿠폰함 → "사용" 버튼**을 눌러야 코인으로 전환됨.
유효기간 내 미사용 시 자동 소멸.

#### 쿠폰 종류별 경로

| 종류 | 경로 | 특징 |
|------|------|------|
| 회원가입 쿠폰 | 가입 완료 시 자동 발급 | 별도 설정 없으면 발급 안 됨 (쿠폰존 등록 필요) |
| 보유 쿠폰 사용 | 이미 가진 쿠폰을 사용 버튼으로 전환 | 만료일 확인 필수 |
| 코드 입력 쿠폰 | 마이페이지 → 쿠폰 코드 입력 | SNS 이벤트·제휴 등에 활용 |
| 30일 출석 달성 쿠폰 | 출석 30일 달성 시 자동 발급 | 14일 만료 |

> **현재 상태**: 쿠폰존이 등록되어 있지 않아 회원가입 쿠폰은 현재 비활성 상태.
> 쿠폰 지급을 원하면 **매출현황 → 쿠폰존 관리**에서 신규 쿠폰존 생성 필요.

---

### 3. 후기 작성 보상

상담 후 후기를 작성하면 자동 코인 지급. 최소 사용 코인 조건 없음 (금액에 상관없이 후기 작성 시 지급).

| 후기 종류 | 지급 코인 | 비고 |
|---------|---------|------|
| 일반 후기 작성 | **500 코인** | 텍스트만 작성해도 지급 |
| 사진 포함 후기 | **1,000 코인** | 기본 500 + 사진 보너스 500 |

- 후기 작성은 **상담 1회당 1번**만 가능
- 관리자 설정: **환경설정 → 일반환경설정 → 후기 포인트** 섹션에서 금액 변경 가능
- `payout_enabled=0` 으로 설정하면 후기 코인 지급 완전 중단

---

### 4. 관리자 베스트 후기 선정

관리자가 특정 후기를 베스트로 선정하면 작성자에게 10,000코인이 즉시 지급됨.

- 금액: **10,000 코인** (코드에 고정, 변경 불가)
- 베스트 해제해도 지급된 코인 회수 없음
- 경로: **상담관리 → 상담후기 관리** → 후기 우측 "베스트" 버튼 클릭

---

### 5. 관리자 수동 지급

이벤트 보상, 시스템 오류 보정 등 직접 코인을 지급할 때 사용.

- 경로: **회원현황 → 고객 리스트** → 해당 회원 클릭 → 포인트 조정
- 금액: 자유 설정 (양수 = 지급, 음수 = 차감)
- 사유 입력 필수 (관리자 감사 로그에 기록됨)

---

### 6. 회원가입 포인트 ⏳ (출시 예정)

가입 완료 시 자동으로 코인을 직접 지급하는 기능. 현재는 설정 화면만 있고 미구현 상태.
출시 후에는 **환경설정 → 일반환경설정 → 회원 섹션**에서 금액 설정 가능.

---

### 7. 로그인 포인트 ⏳ (출시 예정)

로그인할 때마다 코인을 지급하는 기능. 현재는 설정 화면만 있고 미구현 상태.
출시 후에는 **환경설정 → 일반환경설정 → 회원 섹션**에서 금액 설정 가능.

---

## 이런 상황을 보셨나요?

### 회원이 "코인이 생겼는데 뭔지 모르겠어요"

→ 회원 마이페이지 → 코인 내역에서 `rel_action` 라벨 확인:
- `coupon_use` = 쿠폰 사용
- `coupon_redeem` = 쿠폰 코드 입력
- `admin_adjust` = 관리자 수동 지급
- `review_best:N` = 베스트 후기 선정

### "출석 30일 했는데 쿠폰이 안 왔어요"

→ 마이페이지 → 내 쿠폰함 확인. 쿠폰으로 발급되므로 **"사용" 버튼을 눌러야** 코인 전환됨.

### "후기 작성했는데 코인이 안 들어왔어요"

→ 환경설정에서 `payout_enabled` 가 `1` 인지 확인. `0` 이면 후기 코인 미지급.

### "관리자가 베스트 후기 해제했는데 코인 돌려받을 수 있나요?"

→ 불가. 선정 시 즉시 지급, 해제해도 회수 없음. 운영 정책으로 미리 안내 필요.

## 운영자가 설정하는 위치

| 항목 | 관리자 메뉴 위치 |
|------|--------------|
| 출석 보상 금액/활성화 | 회원현황 → 출석 관리 → 정책 설정 탭 |
| 후기 코인 금액/활성화 | 환경설정 → 일반환경설정 → 후기 포인트 섹션 |
| 쿠폰존 생성/관리 | 매출현황 → 쿠폰존 관리 |
| 베스트 후기 선정 | 상담관리 → 상담후기 관리 |
| 개별 회원 수동 지급 | 회원현황 → 고객 리스트 → 회원 상세 → 포인트 조정 |
