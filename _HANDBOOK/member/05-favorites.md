# 단골 / 즐겨찾기 (favorites)

## 한 줄로 답하면
**회원이 마음에 드는 상담사를 단골 등록하면 단골 페이지에서 빠른 접근. 단골 상담사 접속 시 회원에게 알림톡 (옵션).**

## 동작

```
회원이 상담사 카드 → 하트 아이콘 클릭
  ↓
사주플랜에 단골 등록
  ↓
회원 마이페이지 → 단골 페이지에 표시
  ↓
단골 상담사 접속 시 알림톡 (옵션, 설정 가능)
```

## 이런 상황을 보셨나요?

### 회원이 "단골 등록 안 돼요" 라고 클레임

→ 이미 등록되어 있을 가능성. 하트가 핑크색이면 단골. 흰색이면 미등록.

### "단골 상담사 접속 알림이 너무 자주 와요"

→ 정책상 일일 1회 제한 (또는 회원이 알림 끄기 가능). 옵션 검토.

### "단골 해제했는데 알림이 또 와요"

→ 캐시 또는 시스템 지연 가능. 1시간 이상 지속 시 점검.

## 회원이 자주 묻는 질문

### Q. "단골 등록 한도 있나요?"

정책상 무제한. 다만 너무 많으면 단골 페이지 정렬 어려울 수 있음.

### Q. "단골 상담사 알림 끄려면?"

회원 마이페이지 → 설정 (또는 알림 설정) 에서 끄기 가능.

## 이건 정상인가요? 에러인가요?

| 상황 | 답변 |
|---|---|
| 회원이 하트 클릭 → 단골 등록 + 핑크색 변경 | ✅ 정상 |
| 하트 다시 클릭 → 단골 해제 | ✅ 정상 |
| 단골 상담사 접속 → 알림톡 (회원에게) | ✅ 정상 |
| 같은 상담사 중복 등록 시도 | ❌ 차단 (UNIQUE) |
| 단골 해제했는데 단골 페이지에 남음 | ⚠️ 캐시. 새로고침 |

## 운영자가 확인하는 위치

| 확인 항목 | 메뉴 위치 |
|---|---|
| 회원별 단골 상담사 목록 | 사이드바 → 회원현황 → 고객 리스트 → 상세 |
| 단골 통계 (인기 상담사) | 별도 통계 (개발 가능) |
