# 출석 보상 시스템 (일일 코인·연속 보너스·30일 쿠폰)

## 한 줄로 답하면
**회원이 매일 1회 출석하면 무료 코인 적립. 연속 출석 5·10·15·20일에 추가 보너스. 30일 달성 시 10,000코인 쿠폰. 가입 후 3일 경과해야 시작 가능.**

---

## 왜 출석 보상을 만들었나요?

1. **DAU(일일 활성 사용자) 향상** — 매일 앱을 열게 만드는 가장 단순하고 효과적인 방법입니다.
2. **습관 형성** — 연속 보너스가 있으면 하루 빠지면 아깝다는 심리가 생겨 더 자주 접속합니다.
3. **첫 상담 유도** — 무료 코인이 쌓이면 회원이 실제 상담을 시도할 확률이 높아집니다.
4. **상담사 분리** — 상담사도 출석 보상이 있지만 회원의 절반 수준입니다. 상담사는 이미 수익이 있으므로 별도의 큰 인센티브가 불필요합니다.

## 현재 설정값

| 상황 | 지급 코인 | 당일 총 수령 |
|------|---------|-----------|
| 매일 출석 (기본) | **+100 코인** | 100 |
| 연속 5일 달성 | **+200 추가** | 300 |
| 연속 10일 달성 | **+300 추가** | 400 |
| 연속 15일 달성 | **+400 추가** | 500 |
| 연속 20일 달성 | **+500 추가** | 600 |
| 30일 달성 | **10,000코인 쿠폰** | 쿠폰 발급 (14일 유효) |

### 상담사 출석 보상 (회원의 절반)

| 상황 | 지급 코인 |
|------|---------|
| 매일 출석 (기본) | **+50 코인** |
| 연속 5일 달성 | **+100 추가** |
| 연속 10일 달성 | **+150 추가** |
| 연속 15일 달성 | **+200 추가** |
| 연속 20일 달성 | **+250 추가** |
| 30일 달성 | 없음 |

### 기타 설정
- **가입 후 최소 3일 경과** 해야 출석 가능 (어뷰징 방지)
- **일일 총 한도**: 회원 1,000,000코인 / 상담사 500,000코인 (비정상 대량 지급 방지)
- **IP 일일 제한**: 동일 IP 과다 출석 차단 (동일 환경에서 여러 계정 어뷰징 방지)
- 30일 쿠폰 유효기간: **14일**

## 연속 출석이란?

- 오늘 출석 + 어제 출석 = 연속 2일
- 하루라도 빠지면 **연속일 초기화** (0부터 재시작)
- 자정 KST 기준으로 날짜 계산

## 경우의 수별 설명

### Q. "출석 보상이 안 들어왔어요 (회원 클레임)"
→ 확인 순서:
1. 해당 회원 출석 이력 확인 (고객 리스트 → 출석 탭)
2. 오늘 이미 출석했는지 확인
3. 가입 후 3일이 지났는지 확인
4. 출석 정책 활성화 여부 확인 (출석 관리 → 정책 탭)

### Q. "연속 5일 출석했는데 보너스가 없어요"
→ 연속 출석 보너스는 운영자 설정값. 출석 관리 → 정책 탭에서 day5_bonus 값 확인.
현재: 회원 +200코인, 상담사 +100코인

### Q. "30일 쿠폰을 받았는데 코인이 안 들어왔어요"
→ 30일 달성 쿠폰은 **마이페이지 → 내 쿠폰함에서 '사용' 버튼**을 눌러야 코인으로 전환됩니다.
쿠폰 유효기간 14일 이내에 사용해야 합니다.

### Q. "자정 직후 (00:01) 출석했는데 어제로 인정됐어요"
→ 사주플랜은 KST 자정 기준. 00:01 = 새 날이므로 오늘 출석. 어제로 인정됐다면 사고.

### Q. "출석 보상 금액을 바꾸고 싶어요"
→ 회원현황 → 출석 관리 → 정책 설정 탭에서 day1, day5_bonus 등 값 변경

## 이건 정상인가요? 에러인가요?

| 상황 | 판단 |
|------|------|
| 매일 1회만 출석 가능 | ✅ 정상 |
| 자정 KST 기준 날짜 리셋 | ✅ 정상 |
| 출석 보상이 free_balance로 적립 | ✅ 정상 (무료 코인) |
| 출석 보상이 paid_balance로 적립 | ❌ 정책 위반 — 즉시 확인 |
| 가입 3일 미경과 회원에게 보상 | ❌ 어뷰징 방지 정책 위반 |
| 같은 IP에서 여러 계정 출석 | ⚠️ IP 한도 정책 확인 필요 |
