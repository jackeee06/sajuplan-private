# 회원이 충전하면 어떻게 진행되나요?

## 한 줄로 답하면
**회원이 카드/계좌로 결제 → 사주플랜과 m2net 양쪽에 동시 적립 → 회원에게 안내 알림톡 발송.**

## 충전 흐름

```
회원이 충전 페이지 → 금액 선택 → 결제 방법 (카드/가상계좌/간편결제)
  ↓
PG (AG9) 카드 승인 또는 가상계좌 입금 확인
  ↓
사주플랜:
  1. 회원 코인 적립 (paid_balance += 충전 금액)
  2. m2net 측에도 동시 적립 (통화·채팅 차감용)
  3. 결제 이력 기록
  4. 회원에게 알림톡 "결제 확인" 발송
  ↓
회원 마이페이지에서 즉시 잔액 확인 가능
```

## 결제 방법

| 종류 | 처리 |
|---|---|
| **카드 결제** | 즉시 승인 → 즉시 적립 |
| **가상계좌** | 가상계좌 발급 → 회원이 입금 → 입금 확인 시 적립 |
| **간편결제** (페이코 등) | 즉시 승인 → 즉시 적립 |
| **사주플랜페이 자동충전** | 회원이 카드 등록 후 → 잔액 부족 시 자동 결제 (운영 시작 후 활성화) |

## 이런 상황을 보셨나요?

### 회원이 "결제했는데 코인이 안 들어왔어요" 라고 클레임

→ 가능 원인:
1. PG (AG9) 응답 지연 — 1-2분 안에 자동 처리됨
2. 가상계좌 입금이 늦었음 (회원이 입금만 하고 시간 차이)
3. 결제 실패 (카드 한도 등)

→ 답변: "잠시만 기다려주세요. 자동으로 처리됩니다. 5분 지나도 안 들어오면 결제 내역과 함께 알려주세요." 그 후 사이드바 → 매출현황 → 결제 내역 에서 확인.

### "충전 후 사주플랜 잔액과 m2net 측 잔액이 달라요"

→ 일시적 어긋남 가능. 사주플랜 자동 정정 시스템이 30초 안에 동기화. 1시간 지나도 어긋나면 점검 필요.

### "가상계좌 입금했는데 적립 안 됐어요"

→ 입금 확인 자동 처리 — 보통 5-10분 내. 회원에게 입금 시각과 사용 은행 확인 후 결제 내역 검토.

## 회원이 자주 묻는 질문

### Q. "결제 영수증 어디서 받아요?"

회원 → 마이페이지 → 결제 내역 → 영수증 다운로드 또는 인쇄.

### Q. "충전한 코인 환불 받을 수 있나요?"

가능. 단 사용한 코인은 환불 X. 미사용 잔액만 환불 가능. 운영자가 사이드바 → 매출현황 → 환불 이력 에서 수동 처리.

### Q. "자동충전 설정은 어떻게?"

회원 → 마이페이지 → 충전 → 자동충전 등록. 카드 등록 후 잔액 임계값 이하 시 자동 결제.

## 이건 정상인가요? 에러인가요?

| 상황 | 답변 |
|---|---|
| 카드 결제 → 즉시 적립 | ✅ 정상 |
| 가상계좌 → 입금 후 5분 안에 적립 | ✅ 정상 |
| 결제 성공인데 1시간 후에도 적립 X | ❌ **사고**. 결제 내역 확인 + 수동 처리 |
| 사주플랜과 m2net 잔액 1시간+ 어긋남 | ❌ **사고**. 자동 정정 실패 |
| 회원이 받은 영수증 vs 사주플랜 적립 금액 다름 | ❌ **사고**. 즉시 확인 + 정정 |

## 운영자가 확인하는 위치

| 확인 항목 | 메뉴 위치 |
|---|---|
| 결제 내역 | 사이드바 → 매출현황 → 결제 내역 |
| 회원 잔액 | 사이드바 → 회원현황 → 고객 리스트 → 상세 |
| 충전금액 설정 (옵션) | 사이드바 → 매출현황 → 충전금액 설정 |
| 환불 처리 | 사이드바 → 매출현황 → 환불 이력 |
| 포인트 수동 정정 | 사이드바 → 매출현황 → 포인트 관리 |
