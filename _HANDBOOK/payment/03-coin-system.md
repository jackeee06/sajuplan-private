# 코인 / 수익금 / 포인트 — 용어 어떻게 쓰나요?

## 한 줄로 답하면
**사용자(회원) 화면 = "코인" 통일, 상담사 = "수익금" 통일. 시스템 내부에서는 옛 용어 (포인트, 잔액) 그대로지만 보이는 부분은 두 단어만.**

## 정책 (2026-05-25 결정)

### 회원 영역 = "코인"
- ✅ 코인 / 보유 코인 / 충전 코인 / 사용 코인 / 코인 내역
- ❌ "포인트", "P" 단위 표기 금지
- 표기: `30,000 코인`, `+30,000 코인`, `10,000 코인 사용`

### 상담사 영역 = "수익금"
- ✅ 수익금 / 수익금 내역 / 누적 수익금 / 수익금 정산
- ❌ "수익 포인트", "수익" 단독, "수익 내역" (애매)
- 표기: `1,000,000원` (정산은 통장 입금)

### 관리자 / 사장님 영역
- "매출 / 영업이익 / 영업이익률 / 상담마진 / 회사몫" 등 회계 용어 사용 가능 (사장님 영역)
- 일반 회원 / 상담사에게는 노출 X

## 회원 코인 구조 (3종)

| 종류 | 출처 |
|---|---|
| **유료 코인** (paid) | 회원이 결제로 충전한 코인 |
| **무료 코인** (free) | 출석 보상, 이벤트, 추천 보상 |
| **수익 코인** (earning) | 상담사가 받은 수익 (정산 전 누적) |

## 차감 우선순위

채팅·통화 사용 시:
1. **유료 코인 먼저** 차감
2. 유료 0 되면 **무료 코인** 차감
3. 둘 다 0 되면 통화/채팅 종료

→ 회원이 무료 코인을 더 오래 보유 (이벤트 보상 보호).

## 이런 상황을 보셨나요?

### "회원 화면에 '포인트 30,000P' 라고 나와요" 라고 제보

→ ❌ **정책 위반**. 사용자 화면은 "코인 30,000" 으로 통일. 디자인 수정 필요.

### "상담사 마이페이지에 '수익 1,000,000원' 이라고 나와요"

→ ⚠️ "수익금 1,000,000원" 으로 통일. "수익" 단독은 모호.

### 회원이 "내 충전 코인 어디로 갔어요?"

→ 회원 마이페이지 → 포인트 (코인 내역) 페이지에서 차감 이력 조회 가능.

## 이건 정상인가요? 에러인가요?

| 상황 | 답변 |
|---|---|
| 회원이 채팅 후 유료 코인부터 차감 | ✅ 정상 정책 |
| 회원이 출석 보상으로 무료 코인 적립 | ✅ 정상 |
| 회원이 추천인 보상 = 무료 코인 적립 | ✅ 정상 (메모리 박제 정책) |
| 사용자 화면에 "포인트" 표기 | ❌ 위반 |
| 상담사 영역에 "수익" 단독 표기 | ⚠️ 모호. "수익금" 으로 |
| 매출이 회원에게 노출 | ❌ 위반. 사장님 영역만 |

## 운영자가 확인하는 위치

| 확인 항목 | 메뉴 위치 |
|---|---|
| 회원 코인 잔액 | 사이드바 → 회원현황 → 고객 리스트 → 상세 |
| 회원 코인 변동 이력 | 사이드바 → 매출현황 → 포인트 관리 |
| 상담사 수익금 누적 | 사이드바 → 회원현황 → 상담사 리스트 → 상세 |
| 코인 수동 정정 | 사이드바 → 매출현황 → 포인트 관리 |
