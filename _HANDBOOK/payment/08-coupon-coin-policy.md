# 쿠폰 & 무료코인 정책

> 확정일: 2026-06-07 | 원칙: 같은 이벤트에 쿠폰·코인 중복 지급 금지

---

## 핵심 원칙

**쿠폰과 즉시 코인은 같은 이벤트에 동시에 사용하지 않는다.**
- 각 이벤트는 "즉시 코인 지급" 또는 "쿠폰 경유" 중 하나만 사용

---

## 즉시 코인 지급 이벤트 (쿠폰함 미노출)

| 이벤트 | 금액 설정 위치 | 비고 |
|---|---|---|
| 회원가입 | setting.member.register_point | 0이면 미지급 |
| 로그인 | setting.member.login_point | 하루 1회 |
| 출석 매일 | setting.attendance.user.day1 | 자동 체크인 |
| 출석 보너스 (5/10/15/20일) | setting.attendance.user.dayN_bonus | 연속 달성 시 |
| 출석 30일 | setting.attendance.user.day30_coupon_amount | 현재 코인 지급 (쿠폰 전환 예정) |
| 후기 작성 | 관리자 설정 | 사진 포함 시 추가 |
| 베스트 후기 선정 | 10,000코인 고정 | 관리자 지정 시 1회 |

---

## 쿠폰 경유 지급 (쿠폰함에 들어감)

| 종류 | 발급 방법 | 현황 |
|---|---|---|
| 코드입력 쿠폰 (cz_type=3) | 쿠폰존관리 등록 → 회원 코드 직접 입력 | 현재 미운영 |
| 관리자 수동 발급 | 쿠폰존관리 → 특정 회원 지정 발급 | 운영 중 |

**관리자 수동 발급 시 주의**: 쿠폰 이름을 반드시 사용자 친화적으로 작성 (내부 용어 금지)

---

## 폐지 이력

| 항목 | 폐지일 | 사유 |
|---|---|---|
| 회원가입 쿠폰 (issueSignupCoupon) | 2026-06-07 | register_point(즉시 코인)와 이중지급 위험 |

처리: 쿠폰존 is_active=false, 미사용 쿠폰 17건 hidden_at 마킹, 코드 호출 제거

---

## 회원 쿠폰 화면 구조

- **쿠폰함 탭**: 사용 가능한 쿠폰 (관리자 수동 발급 또는 코드입력만 나타남)
- **사용내역 탭**: 사용 완료된 쿠폰 히스토리
- 즉시 코인(출석/후기/회원가입)은 쿠폰함에 절대 나타나지 않음
