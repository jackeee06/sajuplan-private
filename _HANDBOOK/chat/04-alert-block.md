# 회원이 채팅 중이면 다른 알림톡이 안 가는 이유

## 한 줄로 답하면
**채팅 진행 중 사용자에게는 다른 회원의 새 채팅 요청 외 모든 알림톡이 자동 차단됩니다. 채팅 흐름 보호 목적이에요.**

## 이런 상황을 보셨나요?

### 회원이 "채팅 중에 답변 알림톡이 안 왔어요" 라고 클레임

→ **의도된 동작입니다.** 회원이 채팅 중일 땐 다른 알림톡 (질문 답변, 쿠폰 발급 등) 이 자동 차단됩니다. 채팅 끝나고 마이페이지에서 확인 가능.

→ 답변: "채팅 진행 중에는 다른 알림이 방해되지 않도록 자동 차단됩니다. 채팅 끝나고 마이페이지 → 알림 에서 확인하실 수 있어요."

### 상담사가 "다른 회원 채팅 요청 알림톡은 채팅 중에도 오는데, 이건 정상인가요?" 라고 문의

→ **정상입니다.** 다른 회원의 새 채팅 요청만 예외적으로 통과됩니다. 상담사가 다음 손님 받을 준비를 할 수 있도록.

### "쿠폰 발급했는데 회원이 못 받았다고 해요"

→ 회원이 채팅 중이었으면 알림톡 차단됨. 채팅 끝나면 마이페이지에서 쿠폰 확인 가능 (정보 자체는 사주플랜에 살아있음).

## 차단 vs 통과 정리

### 통과 (1가지)
- **다른 회원의 새 채팅 요청** — 상담사가 받음 (가게 안 비우기)

### 차단 (그 외 모두)
- 질문 답변 알림 (회원에게 가는 것)
- 후기 작성 요청
- 쿠폰 발급 안내
- 결제 / 충전 확인
- 정산 완료 (상담사용)
- 선지급 처리 (상담사용)
- 운영 알림 (사장님 본인이 채팅 중일 때)

→ 차단된 알림 정보는 **모두 인앱 (마이페이지) 에서 확인 가능**. 정보 손실 없음.

## 회원이 자주 묻는 질문

### Q. "왜 채팅 중에는 알림을 막아요?"

채팅 = 유료 시간 차감 중. 다른 알림이 진동/벨소리로 와서 집중이 깨지면 회원에게 손해. 보호 정책.

→ 답변: "채팅 중 다른 알림이 방해되지 않도록 자동 차단됩니다. 끝나고 마이페이지에서 일괄 확인 가능합니다."

### Q. "차단된 알림은 어디서 봐요?"

마이페이지 → 알림 (상단 종 모양 아이콘) 에서 받은 모든 알림 확인 가능. 알림톡은 못 와도 인앱 알림은 살아있어요.

### Q. "운영자(사장님) 본인도 채팅 중이면 운영 알림 안 오나요?"

네. 사장님이 채팅 중일 때는 매일 운영 요약도 차단됩니다. 채팅 끝나고 마이페이지에서 확인.

## 이건 정상인가요? 에러인가요?

| 상황 | 답변 |
|---|---|
| 채팅 중 회원이 답변 알림톡 못 받음 | ✅ 정상 (차단 정책) |
| 채팅 중 상담사가 다른 회원 채팅 요청 받음 | ✅ 정상 (화이트리스트 통과) |
| 채팅 끝났는데도 알림톡 계속 안 옴 | ⚠️ 차단 정책 외 사유. BizM 발송 사고 또는 회원 폰 차단 검토 |
| 운영자가 매일 일일 요약 알림톡 못 받음 | ⚠️ 채팅 중이면 정상. 그 외 시간엔 미수신 시 점검 |
| 차단된 알림 정보가 마이페이지에도 안 보임 | ❌ **사고**. 인앱 알림 시스템 점검 필요 |

## 운영자가 확인하는 위치

| 확인 항목 | 메뉴 위치 |
|---|---|
| 알림톡 발송 이력 | 사이드바 → 알림 → 알림톡 발송 |
| 차단된 알림 추적 | 시스템 로그 (개발자 도구) |
| 회원의 인앱 알림 (대신 확인) | 사이드바 → 회원현황 → 고객 리스트 → 회원 상세 |

## 정책을 바꾸고 싶다면

- **운영자 알림은 채팅 중에도 통과**: 가능. 사장님이 채팅 중에도 사고 알림은 받고 싶다면.
- **모든 알림 통과 (차단 정책 폐기)**: 권장 X. 채팅 흐름 방해.
- **FCM 푸시도 같이 차단**: 백로그. 코드 분산 작업 큼.

변경 시 사장님께 보고 후 시스템 수정.
