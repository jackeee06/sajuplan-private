# 채팅 끝나기 5분 전 알림

## 한 줄로 답하면
**채팅 잔여 시간이 5분 남으면 회원·상담사 양쪽에 자동 안내. 회원은 충전하기 / 상담사는 마무리 안내.**

## 동작

```
회원이 30분 채팅 중
  ↓
잔여 5분 도달
  ↓
사주플랜이 채팅창에 안내 표시:
  - 회원 화면: "5분 남았어요. 충전하시면 끊김 없이 계속 가능합니다." + [충전하기] 버튼
  - 상담사 화면: "회원 5분 남았어요. 마무리 안내 부탁드립니다." + [확인] 버튼
  ↓
TTS (말소리) + 진동 1회
  ↓
잔여 0 도달 → 자동 종료
```

## 이런 상황을 보셨나요?

### 회원이 "5분 남았다고 알림 왔는데 충전 안 했어요. 채팅 끊기지 않게 해주세요" 라고 클레임

→ 회원이 선택한 시간이 다 되면 자동 종료됩니다. 충전 안 하면 끊김.

→ 답변: "선택하신 시간이 다 되어 자동 종료되었습니다. 추가 대화는 새로 채팅 신청해주세요."

### "5분 알림이 너무 일찍 와요" 또는 "안 왔어요" 라고 클레임

→ 5분 알림은 채팅 진행 중 잔여 시간 5분 도달 시 1회. 채팅 끝 (잔여 0) 보다 5분 일찍.

→ 못 받았다면 시스템 사고 가능. 채팅 시간 + 알림 시각 확인 필요.

## 회원이 자주 묻는 질문

### Q. "5분 알림 받고 충전했는데도 끊겼어요"

충전이 사주플랜에 반영되는 데 시간 차이가 있을 수 있어요. 일반적으로 즉시 반영되지만, 채팅 끝나기 직전 충전은 타이밍 문제 가능.

→ 답변: "충전 시각과 채팅 종료 시각을 확인하겠습니다. 시간 차이로 끊긴 경우 추가 채팅으로 안내드릴게요."

### Q. "왜 5분이에요? 10분 전에 알려주면 더 좋을 텐데"

5분 = 충전 결정·진행에 충분한 시간. 너무 일찍 알리면 회원이 잊고, 너무 늦으면 충전 못 함. 균형점.

## 이건 정상인가요? 에러인가요?

| 상황 | 답변 |
|---|---|
| 5분 남았을 때 양쪽에 알림 + 충전 버튼 | ✅ 정상 |
| 알림 후 채팅 정상 진행 (충전 안 함) | ✅ 정상. 잔여 0 도달 시 끊김 |
| 5분 알림 2회 이상 도착 | ❌ **사고**. 시스템에 표시 1회 정책 |
| 회원에게는 왔는데 상담사에게는 안 옴 | ⚠️ 상담사 화면 폴링 사고. 점검 |
| 채팅 시작 직후 (4분 진행) 에 5분 알림 | ❌ **사고**. 잔여 시간 계산 오류 |

## 운영자가 확인하는 위치

| 확인 항목 | 메뉴 위치 |
|---|---|
| 채팅 시작·종료 시간 | 사이드바 → 상담관리 → 채팅내역 리스트 → 상세 |
| 충전 이력 (5분 알림 받고 충전한 경우) | 사이드바 → 매출현황 → 결제 내역 |
