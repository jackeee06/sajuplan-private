# 상담사가 새 채팅 요청 받기 (4단 안전망)

## 한 줄로 답하면
**회원이 채팅 신청하면 사주플랜이 상담사에게 알림톡 + FCM 푸시 + 앱 안 모달/배너/리스트로 안내합니다. 앱을 꺼놨어도 푸시·알림톡이 도달합니다.**

## 사주플랜의 4단 안전망

### 1. 알림톡 (외부 도달)
- 회원이 채팅 신청 → 상담사 카톡으로 알림톡 도착
- 클릭 시 사주플랜 앱 자동 호출 (APK scheme 등록 시점부터)

### 2. FCM 푸시 (앱 꺼져 있어도 도달) — 2026-06-08 추가
- 회원이 채팅 신청 → 상담사 휴대폰에 푸시 알림
- **앱을 백그라운드로 내리거나 완전히 닫아도 도달** (인앱 모달의 폴링 한계 보완)
- "OO 님이 채팅상담을 신청했습니다. 3분 안에 입장해주세요."
- 상담사 FCM 토큰이 등록돼 있어야 함 (앱 1회 이상 실행 시 자동 등록)

### 3. 앱 안 자동 모달 (앱 포그라운드일 때)
- 상담사가 앱을 **화면에 켜놓고 있을 때만** 5초 폴링으로 감지 → 자동 모달
- 앱이 백그라운드면 OS가 폴링을 멈추므로 안 뜸 → 이때는 2번 푸시가 백업
- "지금 응답하기" 클릭 → 채팅방 진입

### 4. 핑크 배너 (Home 통합 안내)
- N건 (1건 이상) 채팅 요청이 쌓이면 Home 상단 핑크 배너 표시
- "💬 새 채팅 요청 N건" 클릭 → 리스트 페이지

### + 리스트 페이지 (다건 처리)
- 사이드바 (모바일은 메뉴) → 상담사 마이페이지 → 새 채팅 요청
- 오래 기다린 순으로 정렬
- "곧 만료" 배지 (30초 이내) 로 우선 응대 가능

## 이런 상황을 보셨나요?

### "상담사가 채팅 요청을 못 봤어요" 라고 클레임

→ 3단 안전망 (알림톡 + 모달 + 배너) 중 어디서도 못 봤다면 시스템 사고. 또는 상담사가 앱·카톡을 안 본 상태.

→ 답변: "알림톡과 앱 안 안내 모두 동작합니다. 카톡 차단되어 있거나 앱 미진입 상태였을 가능성이 큽니다."

### "상담사가 채팅 중인데 다른 회원의 요청은 어떻게 알아요?"

→ **채팅 중에도 알림톡은 통과** (화이트리스트 정책). 상담사가 다음 손님 받을 준비 가능.

### "리스트 페이지에 본인 채팅 요청이 안 보여요"

→ 다른 상담사에게 매칭된 채팅이거나, 이미 응답한 채팅일 수 있어요. 상담사 본인에게 매칭된 진행 대기 (3분 이내) 만 표시.

## 이건 정상인가요? 에러인가요?

| 상황 | 답변 |
|---|---|
| 채팅 요청 → 즉시 알림톡 + Home 배너 N건 표시 | ✅ 정상 |
| 채팅 1건 발생 → 자동 모달 노출 | ✅ 정상 |
| 알림톡·푸시는 왔는데 앱 안 모달 안 보임 | ⚠️ 정상 가능. 인앱 모달은 앱 포그라운드일 때만 폴링 동작. 앱 닫혀 있었으면 푸시·알림톡으로 도달 |
| 회원에게 incoming 배너 보임 | ❌ **사고**. 상담사 전용. 권한 가드 점검 |
| 알림톡 안 옴 (BizM 응답 정상인데) | ⚠️ 상담사 폰 차단 또는 070 차단. 회원에게 안내 |
| 본인 채팅 요청인데 리스트에 없음 | ⚠️ 3분 자동 취소 됐을 가능성 (오래된 요청). 또는 정합성 점검 |

## 운영자가 확인하는 위치

| 확인 항목 | 메뉴 위치 |
|---|---|
| 상담사별 응답률 | 사이드바 → 회원현황 → 상담사 리스트 → 상세 |
| 자동 취소된 채팅 | 사이드바 → 상담관리 → 채팅내역 리스트 → 종료 사유 |
| 알림톡 발송 이력 | 사이드바 → 알림 → 알림톡 발송 |
