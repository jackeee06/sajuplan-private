# 회원이 채팅 도중 화면을 벗어났어요

## 한 줄로 답하면
**회원이 어떻게 벗어나느냐에 따라 사주플랜이 알아서 안전하게 처리합니다. 코인이 회원 모르게 빠지는 일은 없어요.**

## 이런 상황을 보셨나요?

### 회원이 "채팅 중에 잠깐 다른 앱 봤어요" 라고 클레임

→ **걱정 안 하셔도 됩니다.** 사주플랜은 회원이 채팅방을 벗어나면 자동으로 "잠시 자리비움" 표시를 합니다. 상담사가 그 사이 입장해도 코인은 차감되지 않습니다. 회원이 3분 안에 돌아오면 채팅이 그대로 살아있어요.

### "채팅 신청만 했는데 상담사가 안 와서 그냥 닫았어요" 라고 클레임

→ **차감 0원입니다.** 상담사가 채팅방에 입장하기 전까지는 어떤 경우에도 코인이 빠지지 않습니다 (F 정책). 회원에게 환불할 필요도 없습니다 (애초에 안 빠졌으니까요).

### **상담관리 → 채팅내역 리스트** 에서 "3분 자동 취소" 가 자주 보임

→ **정상입니다.** 회원이 채팅 신청 후 상담사가 3분 안에 응답하지 않거나 회원이 돌아오지 않으면 사주플랜이 자동으로 취소합니다. 회원·상담사 양쪽에 알림톡이 발송되어 알게 됩니다.

## 사주플랜이 처리하는 4가지 경로

회원이 채팅에서 벗어나는 방법은 4가지가 있는데, 사주플랜은 각각 다르게 처리합니다.

### 1. 채팅방 위쪽의 "상담종료" 버튼 클릭
- 정식 종료. 즉시 채팅이 끝납니다.
- 회원에게 "정말 종료할까요?" 확인 모달이 뜬 다음 진행.

### 2. 브라우저 탭을 닫거나 새로고침
- 회원이 의도치 않게 한 행동일 수 있어요.
- 사주플랜은 3분간 기다립니다. 그 안에 돌아오면 채팅 그대로 살아있어요.

### 3. 홈 버튼 누르고 다른 앱 보기
- 잠깐 다른 앱 본 거니까 자동 복귀 처리.
- 돌아오면 채팅 그대로 진행 가능.

### 4. 진짜 3분 넘게 안 돌아옴
- 사주플랜이 자동으로 채팅방 정리 + 양쪽에 알림톡 ("3분간 응답 없어서 자동 취소되었습니다").
- 회원도 상담사도 헛수고 안 하도록 시스템이 알아서 마무리.

## 이건 정상인가요? 에러인가요?

| 회원/상담사 클레임 | 답변 |
|---|---|
| "채팅 5초 했는데 코인 다 빠졌어요" | ⚠️ **에러일 수 있음**. 5초 안에 종료하면 전액 환불되어야 합니다. 환불 안 됐다면 사이드바 → 매출현황 → 환불 이력 에서 수동 환불 처리. |
| "상담사 안 들어왔는데 코인 빠졌어요" | ❌ **사고**. 즉시 사장님께 보고 + 시스템 점검 필요. F 정책 위반. |
| "잠깐 카톡 봤다가 돌아왔는데 채팅 끊겼어요" | ⚠️ 3분 넘으면 정상 자동 취소. 3분 안인데 끊겼으면 시스템 사고. |
| "채팅 신청만 하고 안 들어갔는데 코인 빠짐" | ❌ **사고**. F 정책 위반. 보고 + 점검. |
| "3분 자동 취소됐는데 양쪽에 알림톡 안 왔어요" | ⚠️ 알림톡 발송 사고. [BizM 알림톡 템플릿](alert/02-bizm-templates) 항목 참조. |

## 회원이 자주 묻는 질문 (사장님이 받을 만한 클레임)

### Q. "왜 채팅방을 닫지도 않았는데 끊겼어요?"

회원이 다른 앱 본 시간이 3분 넘었기 때문이에요. 사주플랜은 회원·상담사 모두 보호하기 위해 3분 이상 무응답이면 자동 취소합니다.

→ 답변: "3분 이상 자리비우심으로 자동 종료된 케이스입니다. 다시 채팅 신청하시면 새로 연결됩니다."

### Q. "그러면 회원이 잠깐 다른 앱 봐도 되는 거 아니에요?"

네. 3분 안에 돌아오면 채팅방이 그대로 살아있고, 그 사이 코인도 차감되지 않습니다.

회원이 다른 앱 본 동안 상담사가 입장했다면: 상담사는 회원이 "잠시 자리비움" 표시인 걸 보고 기다리거나 메시지를 남길 수 있어요. 회원이 돌아오면 정상 진행.

### Q. "채팅방 안 들어가고 종료 버튼만 눌렀는데 환불해야 되나요?"

상담사가 입장하기 전 종료라면 **코인이 처음부터 안 빠졌습니다** (F 정책). 환불 처리 필요 없음.

## 운영자가 확인하는 위치

| 확인 항목 | 메뉴 위치 |
|---|---|
| 자동 취소된 채팅 보기 | 사이드바 → 상담관리 → 채팅내역 리스트 |
| 환불 처리 (수동) | 사이드바 → 매출현황 → 환불 이력 |
| 운영 알림 (사고 알림) | 사장님 카톡 알림톡 |

## 정책을 바꾸고 싶다면

- **3분 → 더 길게/짧게** 변경 가능. 다만 너무 짧으면 정상 사용자가 불편, 너무 길면 사주플랜 자원 낭비.
- **F 정책 (상담사 입장 전 차감 0)** 변경은 권장 X. 회원 보호 핵심 정책.

변경 결정 시 사장님께 보고 후 시스템 수정 필요.
