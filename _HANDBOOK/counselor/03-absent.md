# 상담사 자동 부재중 전환

## 한 줄로 답하면
**상담사가 24시간 동안 채팅 응답 안 하면 사주플랜이 자동으로 "부재중" 상태로 전환. 회원 화면에서 그 상담사 안 보임. 상담사 본인에게 알림톡 + 사장님에게 운영 알림.**

## 동작

```
사주플랜이 매 1시간마다 검사:
  - 상담사 마지막 활동 시각이 24시간 전이면 → 부재중 전환
  ↓
- 상담사 알림톡: "자동 부재중으로 전환되었습니다"
- 사장님 알림톡: "[상담사] 자동 부재중 전환"
- 회원 화면에서 그 상담사 숨김
  ↓
상담사가 다시 응답하면 자동 활성 복귀
```

## 이런 상황을 보셨나요?

### 상담사가 "내가 부재중으로 바뀌었어요?" 라고 문의

→ 24시간 응답 안 했으면 자동 처리. 활동 (앱 진입, 채팅·통화 응답) 즉시 복귀.

→ 답변: "사주플랜에 다시 접속하시면 자동으로 활성 상태가 됩니다."

### 사장님에게 "상담사 X 부재중" 운영 알림 자주 옴

→ 정상. 24시간 무응답 상담사 발생 시마다 알림. 잦으면 그 상담사 활동 패턴 검토.

### "휴가인데 사주플랜에 알리고 싶어요"

→ 현재 자동 부재중만. 명시적 휴가 모드는 별도 기능 (향후 검토).

## 회원이 받을 수 있는 클레임

### Q. "어제 자주 봤던 상담사 안 보여요"

→ 그 상담사가 24시간 응답 없어서 자동 부재중. 복귀하면 다시 보임.

## 이건 정상인가요? 에러인가요?

| 상황 | 답변 |
|---|---|
| 24시간 무응답 → 자동 부재중 + 알림 | ✅ 정상 |
| 부재중 상태에서 새 채팅 응답 → 자동 활성 복귀 | ✅ 정상 |
| 자주 부재중 ↔ 활성 반복 | ⚠️ 상담사 활동 패턴 검토 |
| 활성인데 회원 화면에 부재중 표시 | ❌ **사고**. 점검 |
| 부재중 인데 운영자에게 알림 안 옴 | ❌ **사고**. 알림 시스템 점검 |

## 운영자가 확인하는 위치

| 확인 항목 | 메뉴 위치 |
|---|---|
| 부재중 상담사 목록 | 사이드바 → 회원현황 → 상담사 리스트 → 필터 "부재중" |
| 자동 전환 이력 | 사이드바 → 회원현황 → 상담사 리스트 → 상세 → 상태 이력 |
| 강제 활성 복귀 (수동) | 사이드바 → 회원현황 → 상담사 리스트 → 상세 → "활성 복귀" 버튼 |
