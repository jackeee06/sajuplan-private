# 상담사 회원 차단 (블랙리스트)

## 한 줄로 답하면
**상담사가 특정 회원을 차단하면, 그 회원의 앱 상담사 목록에서 해당 상담사가 완전히 사라집니다. 회원은 차단 사실을 모릅니다.**

---

## 왜 이 기능이 필요한가요?

1. **상담사 보호** — 욕설, 괴롭힘, 반복 민원 등 불쾌한 회원으로부터 상담사를 보호합니다.
2. **운영 유연성** — 상담사가 직접 조작하지 않고 관리자를 통해 처리하므로 오남용을 방지합니다.
3. **회원 비인지** — 차단된 회원은 상담사가 목록에서 사라진 것만 보이고 차단 사실을 알 수 없어 분쟁이 줄어듭니다.

---

## 운영 절차

### 차단 요청이 들어올 때

```
상담사가 전화로 요청
  ↓
"몇 시 몇 분에 상담한 사람 차단해 주세요"
  ↓
관리자 → 상담 내역에서 해당 건 찾기
  ↓
해당 행의 [차단] 버튼 클릭 → 사유 입력 → 완료
```

### 방법 1 — 상담 내역에서 바로 차단 (추천, 가장 빠름)

1. **상담관리 → 채팅내역/사용(상담) 내역** 진입
2. 해당 시간대 상담 건 찾기 (회원 이름 또는 상담사 이름으로 검색)
3. 해당 행 맨 오른쪽 **[차단]** 버튼 클릭
4. 사유 입력창 → "욕설" / "괴롭힘" 등 메모 → 확인
5. 완료 알림 → 즉시 적용

### 방법 2 — 상담사 상세 페이지에서 관리

1. **회원현황 → 상담사 리스트** → 해당 상담사 클릭
2. 페이지 하단 **"차단 관리"** 섹션 확인
3. **휴대폰 번호** 입력 (예: `01012345678`) + 사유 입력
4. **[차단 추가]** 버튼 클릭
5. 차단 목록에 해당 회원 추가됨

---

## 차단 해제 방법

1. **회원현황 → 상담사 리스트** → 해당 상담사 클릭
2. 페이지 하단 **"차단 관리"** 섹션
3. 차단 목록에서 해당 회원 오른쪽 **[차단 해제]** 클릭
4. 확인 → 즉시 해제됨

---

## 이런 상황을 보셨나요?

### "상담사가 특정 회원에게 상담을 거부하고 싶다고 해요"
→ 상담 내역 또는 상담사 상세에서 해당 회원을 차단해주세요.

### "차단한 회원이 다른 상담사에게는 상담 가능한가요?"
→ 네. 차단은 **특정 상담사 ↔ 특정 회원** 관계에만 적용됩니다. 다른 상담사에게는 영향 없습니다.

### "차단된 회원이 차단을 알 수 있나요?"
→ 알 수 없습니다. 회원에게는 해당 상담사가 목록에서 조용히 사라질 뿐입니다.

### "상담 진행 중에도 차단이 즉시 적용되나요?"
→ 이미 진행 중인 상담에는 영향 없습니다. **다음 번 접속부터** 해당 상담사가 안 보입니다.

### "차단된 회원을 검색해도 안 나오나요?"
→ 맞습니다. 상담사 목록 조회와 검색 모두에서 차단된 상담사가 숨겨집니다.

---

## 이건 정상인가요? 에러인가요?

| 상황 | 판단 |
|------|------|
| 차단 즉시 회원 목록에서 상담사 사라짐 | ✅ 정상 |
| 차단 후에도 진행 중인 채팅은 계속 됨 | ✅ 정상 (진행 중 상담에는 영향 없음) |
| 해당 상담사가 다른 회원에게도 안 보임 | ❌ 버그 — 즉시 확인 필요 |
| 차단 해제 후에도 여전히 안 보임 | ❌ 버그 — 앱 캐시 갱신 또는 재로그인 유도 |

---

## 운영자가 확인하는 위치

| 작업 | 메뉴 위치 |
|------|---------|
| 상담 내역으로 빠른 차단 | 상담관리 → 사용(상담) 내역 → 해당 행 [차단] |
| 차단 목록 관리·해제 | 회원현황 → 상담사 리스트 → 상담사 상세 → 하단 차단 관리 |
