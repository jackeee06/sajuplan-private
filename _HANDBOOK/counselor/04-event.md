# 이벤트 상담사 시스템 (운영 시작 후 도입 예정)

## 한 줄로 답하면
**신규 상담사를 회원에게 별도 노출하는 기능. 정책 합의됨, 코드 미구현. 운영 시작 후 신규 상담사 유입 데이터 보고 도입 결정.**

## 현재 상태

| 항목 | 상태 |
|---|---|
| 정책 합의 | ✅ |
| DB 설계 | ⏳ 검토 중 |
| 코드 구현 | ❌ 미시작 |
| 운영 시작 | ❌ 운영 시작 후 결정 |

## 계획된 정책

### 이벤트 상담사 = 신규 또는 프로모션 상담사
- 신규 가입 후 첫 X일 동안 "이벤트" 라벨
- 또는 운영자가 프로모션 목적으로 임시 라벨 부여

### 회원 측 노출
- 홈 / 상담사 리스트에서 별도 섹션
- "⭐ NEW" 같은 라벨
- 단가 할인 또는 첫 채팅 무료 (정책 결정 필요)

### 종료 조건
- X일 경과 또는 N건 상담 완료 → 일반 상담사 전환
- 또는 운영자가 수동 종료

## 왜 이런 정책?

운영 시작 초기 = 상담사 풀 작음 → 신규 상담사 유입 필수 → 노출 부스트 + 회원 진입 장벽 ↓.

## 도입 시점 (예상)

- 운영 시작 후 1-2개월
- 신규 가입 상담사 수치 보고 결정
- 일반 상담사 풀 충분하면 안 도입 가능

## 운영자가 확인할 (도입 후)

| 확인 항목 | 메뉴 위치 (예정) |
|---|---|
| 이벤트 상담사 목록 | 사이드바 → 회원현황 → 상담사 리스트 → 필터 "이벤트" |
| 이벤트 상담사 운영 (등록·종료) | 사이드바 → 회원현황 → 상담사 리스트 → 상세 → 이벤트 라벨 |
