# 상담사 등급과 단가 시스템은 어떻게 운영하나요?

## 한 줄로 답하면
**상담사는 당월 누적 상담시간이 다음 등급 임계값에 도달하는 즉시 승급됩니다. 매월 1일에는 전월 실적으로 강등 여부를 평가합니다. 등급이 높을수록 단가 선택지가 넓어지고 정산률도 높아집니다.**

---

## 🆕 실시간 승급 시스템 (2026-06-07 신설)

### 핵심 변화

| 항목 | 기존 (월 1일만) | 변경 후 |
|---|---|---|
| 승급 시점 | 매월 1일 크론 | **당월 누적시간 달성 즉시** |
| 강등 시점 | 매월 1일 크론 | 매월 1일 유지 (변화 없음) |
| 단가 변경 | 매월 1일 해제 | **승급 즉시 해제** |
| 정산률 적용 | 월 단위 단일 등급 | **상담 시작 시점 등급 기준** |

### 동작 예시 (파트너4 → 파트너5)

```
5월 1일: 파트너4 시작 (시간 카운팅 시작)
5월 25일 오후 3시: 당월 누적 120시간 달성
  → 오후 3시 즉시 파트너5 승급
  → 단가 선택지 즉시 파트너5 범위로 확대 (1,200~2,000원)
  → 알림톡 발송: "파트너5로 승급되었습니다!"
  → 5월 25일 오후 3시 이후 상담 = 파트너5 수수료율(70%) 적용
6월 1일: 카운팅 리셋. 6월 누적시간 새로 시작.
6월 1일: 6월의 실적(파트너5 유지 기준 120시간)으로 5월 평가는 이미 통과됨.
6월 30일까지 120시간 미달 → 7월 1일에 파트너4로 강등.
```

### 같은 달에 두 단계 이상 승급 가능

예) 파트너3가 5월 10일에 파트너4 조건(90시간) 달성 → 즉시 파트너4
→ 5월 20일에 파트너5 조건(120시간) 달성 → 즉시 파트너5

이 경우 정산:
- 5/1~5/10: 파트너3 수수료(60%)
- 5/10~5/20: 파트너4 수수료(63%)
- 5/20~5/31: 파트너5 수수료(70%)

## 등급 구조 (현재 설정값)

| 등급 | 코드 | 승급 임계값(당월 누적 상담시간) | 단가 선택지(30초당) | 정산률 |
|------|------|----------------------|-----------------|--------|
| 예비파트너 | preliminary | — (신규 기본) | 800, 1,000원 | **40%** |
| 파트너 1 | partner1 | 20시간 이상 | 800, 1,000원 | **52%** |
| 파트너 2 | partner2 | 40시간 이상 | 1,000, 1,200원 | **56%** |
| 파트너 3 | partner3 | 70시간 이상 | 1,000, 1,200, 1,300원 | **60%** |
| 파트너 4 | partner4 | 90시간 이상 | 1,000, 1,200, 1,300, 1,400, 1,500원 | **63%** |
| 파트너 5 | partner5 | 120시간 이상 | 1,200, 1,300, 1,400, 1,500, 1,800, 2,000원 | **70%** |

> 임계값 기준: **당월 누적 상담 시간** (전화 + 채팅 합산, 매월 1일 초기화).
> 정산률: 회원이 사용한 코인 중 상담사에게 돌아가는 비율.

## 단가란 무엇인가요?

단가 = **30초당 코인**. 상담사가 자신의 등급 내에서 직접 선택합니다.

예시:
- 상담사가 단가 1,000원 선택 → 회원이 30초마다 1,000코인 차감
- 1코인 = 1원이므로 30초당 1,000원, 분당 2,000원

## 승급·강등 규칙

### 승급 (실시간)
- 당월 누적 상담시간이 다음 등급 임계값 도달 → **즉시 자동 승급**
- 같은 달에 여러 단계 승급 가능
- 승급 즉시 단가 변경 락 해제 → 새 등급 단가로 즉시 변경 가능

### 강등 (매월 1일)
- 전월 상담시간이 현재 등급 유지 기준 이하면 강등
- **강등은 한 단계씩만** — 파트너5가 기준 미달이어도 파트너4로만 강등
- 강등 후에도 단가 변경 1회 기회 주어짐

### 시간 카운팅 규칙
- 매월 1일 00:00 KST 에 당월 카운팅 초기화
- 월중 승급 후 초과된 시간은 다음 달로 이월 안 됨 (버림)
  - 예) 5월 25일 승급 후 5/26~31 시간은 6월 카운팅에 포함 안 됨

## 경우의 수별 설명

### Q. "상담사가 이번 달 열심히 해서 목표 시간 채웠어요 — 언제 승급되나요?"
→ 목표 시간을 채운 **그 순간 즉시** 파트너5로 승급됩니다. 1일까지 기다릴 필요 없어요.

### Q. "단가를 올리고 싶어요 (방금 승급했어요)"
→ 승급 즉시 단가 변경 가능합니다. 마이페이지 → 등급/단가 → 변경 버튼.

### Q. "수익금 내역에 등급이 다르게 표시돼요"
→ 정상입니다. 각 상담 건은 **상담 시작 시점의 등급**으로 정산률이 적용됩니다. 
   승급 전 상담과 승급 후 상담이 다른 수수료율로 정산되므로 등급 뱃지가 다르게 표시됩니다.

### Q. "이번 달 진행상황이 어디서 보이나요?"
→ 마이페이지 → 등급/단가 섹션 상단 프로그레스 바:
- 이번 달 누적 시간 / 다음 등급까지 남은 시간
- 이번 달 실시간 승급 이력

### Q. "이번 달 파트너5로 올랐는데 다음 달 기준이 뭐예요?"
→ 파트너5 유지 기준 = 120시간/월. 6월에 120시간 못 채우면 7월에 파트너4로 강등.

## 승급 알림은 어떻게 오나요? (2026-06-07)

목표 시간을 채운 순간 상담사에게 두 가지 경로로 알림이 갑니다.

| 경로 | 앱 상태 | 모양 |
|---|---|---|
| **FCM 포그라운드** | 앱 켜져 있음 | 상단 배너 → 탭하면 마이페이지로 이동 |
| **FCM 백그라운드** | 앱 꺼져 있음 | 시스템 알림 → 탭하면 앱 열리며 마이페이지 |
| **인앱 모달** | 마이페이지 진입 시 | "🎉 파트너5로 승급되었습니다!" 전체화면 팝업 |
| **알림톡** | 언제나 | 카카오톡으로 도달 (BizM 템플릿 등록 후 자동 발송) |

> ⚠️ **사장님 할 일**: BizM 콘솔에서 `counselor_grade_upgraded` 템플릿 등록 + 검수 신청 필요.
> 등록 전까지 FCM + 인앱 모달은 정상 작동, 알림톡만 미발송.

### Q. "승급 알림이 안 왔어요"
→ 확인 순서:
1. 앱이 켜져 있었는지 → 포그라운드 FCM 배너 또는 마이페이지 진입 시 팝업
2. 앱 알림 권한 ON인지 → 설정 → 앱 → 사주플랜 → 알림 허용
3. 마이페이지 진입 후 팝업이 안 뜨면 → DB에 승급 이력 확인 (어드민 → 상담사 등급 상세)

## 이건 정상인가요? 에러인가요?

| 상황 | 판단 |
|------|------|
| 목표시간 달성 순간 즉시 승급 알림 | ✅ 정상 |
| 매월 1일 등급 변동 알림 (강등) | ✅ 정상 (크론 자동 실행) |
| 강등이 2단계 한번에 일어남 | ❌ 정책 위반 — 즉시 확인 |
| 파트너5가 800원 단가 선택 | ❌ 버그 — 파트너5 최저 단가는 1,200원 |
| 신규 상담사가 파트너1로 시작 | ❌ 정책 위반 — 신규는 예비파트너부터 |
| 수익금 내역에 건마다 다른 등급 뱃지 | ✅ 정상 (승급 전/후 상담 구분 표시) |

## 운영자가 확인/변경하는 위치

| 작업 | 메뉴 위치 |
|------|---------|
| 현재 등급 분포 | 회원현황 → 등급 관리 |
| 실시간 승급 이력 | 회원현황 → 등급 관리 → 상담사 선택 → 실시간 승급 이력 탭 |
| 단가·정산률 변경 | 환경설정 → 기본환경설정 → 등급/단가 탭 |
| 임계값 변경 | 환경설정 → 기본환경설정 → 등급/단가 탭 → 임계값 |
| 수동 등급 변경 | 회원현황 → 상담사 리스트 → 해당 상담사 → 등급 상세 |
