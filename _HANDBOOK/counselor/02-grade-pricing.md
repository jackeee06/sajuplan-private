# 상담사 등급과 단가는 어떻게 동작해요?

## 한 줄로 답하면
**상담사를 등급별 (일반·우수·특급) 분류하고 분당 단가를 다르게 적용. 운영자가 등급 변경 시 다음 상담부터 새 단가, 옛 상담은 그대로.**

## 등급 종류

| 등급 | 분당 단가 | 의미 |
|---|---|---|
| 일반 (기본) | 표준 | 신규 또는 일반 상담사 |
| 우수 | + α | 인기 + 활동 우수 |
| 특급 | + β | 경력 + 인기 + 활동 모두 |

→ 정확한 수치는 사이드바 → 회원현황 → 등급 관리 에서 운영자가 설정.

## 핵심 정책

### 단가 적용 시점
- 상담 1건 종료 시 그 시점 등급 단가로 확정
- 후속 등급 변경되어도 옛 상담은 옛 단가 그대로 (소급 X)

→ 정산 정확성 + 분쟁 시 명확한 근거

## 이런 상황을 보셨나요?

### 상담사가 "내 등급 어떻게 올려요?" 라고 문의

→ 운영자가 등급 변경. 인기·활동 누적 + 후기 점수 등 종합 판단.

→ 답변: "활동 누적 보고 정기적으로 검토합니다. 평균 후기 점수 + 채팅 시간 등 기준입니다."

### "등급 올랐는데 옛 채팅도 더 받을 수 있어요?"

→ X. 옛 상담은 그때 단가로 확정. 새 단가는 다음 상담부터.

### "회원이 같은 채팅 시간 다른 코인 빠진 거 같아요"

→ 상담사 등급 다르면 단가 다름. 회원 입장에서는 표시 가능 (예: "특급 상담사 = 분당 X 코인").

## 이건 정상인가요? 에러인가요?

| 상황 | 답변 |
|---|---|
| 신규 상담사 = 일반 등급 기본 | ✅ 정상 |
| 운영자가 등급 승급 후 다음 상담부터 새 단가 | ✅ 정상 |
| 등급 변경 후 옛 상담 단가도 자동 변경 | ❌ 정책 위반 |
| 상담사 본인이 등급 직접 변경 시도 | ❌ 차단 (운영자만) |
| 같은 등급 상담사들이 다른 단가 받음 | ❌ 데이터 사고. 점검 |

## 운영자가 확인하는 위치

| 확인 항목 | 메뉴 위치 |
|---|---|
| 등급별 단가 설정 | 사이드바 → 회원현황 → 등급 관리 |
| 상담사별 현재 등급 | 사이드바 → 회원현황 → 상담사 리스트 → 상세 |
| 등급 변경 이력 | 사이드바 → 회원현황 → 상담사 리스트 → 상세 → 등급 이력 |
