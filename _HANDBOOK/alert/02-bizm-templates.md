# BizM 알림톡 어떤 게 있나요?

## 한 줄로 답하면
**사주플랜이 발송하는 알림톡은 약 20종. 채팅 요청 / 결제 확인 / 후기 요청 / 정산 / 운영 알림 등. BizM 콘솔에 사장님이 직접 관리.**

## 이런 상황을 보셨나요?

### "회원이 알림톡 안 왔다고 해요"

→ 가능 원인:
1. 회원이 채팅 중 (자동 차단)
2. 회원 폰이 070 차단
3. 회원 폰이 BizM 발신번호 차단
4. BizM 측 사고

→ 답변: "마이페이지 → 알림 에서 확인 가능합니다. 알림톡 못 받으셔도 사주플랜 앱 안에는 정보 살아있어요."

### "알림톡 본문 바꾸고 싶어요"

→ BizM 콘솔에서 사장님이 직접 수정 → 카카오 재검수 → 통과 후 적용.

### "검수 통과 안 되는 템플릿 있어요"

→ 카카오 정책 (광고성 표현, 과도한 이모지 등) 위반. 본문 다듬어 재신청.

## 이건 정상인가요? 에러인가요?

| 상황 | 답변 |
|---|---|
| 알림톡 발송 성공 | ✅ 정상 |
| 발송 실패 (`K104:TemplateNotFound`) | ❌ 템플릿 검수 미통과 또는 DB 미등록. BizM 콘솔 확인 |
| 발송 실패 (`M107:DeniedSenderNumber`) | ❌ 발신번호 BizM 미등록. 발신번호 등록 |
| 채팅 중 차단 (`recipient_in_chat`) | ✅ 정책 |
| 회원 폰 차단 | ⚠️ 외부 (운영자 조치 X) |

## 운영자가 확인하는 위치

| 확인 항목 | 메뉴 위치 |
|---|---|
| 알림톡 발송 이력 | 사이드바 → 알림 → 알림톡 발송 |
| 알림톡 템플릿 목록 | 사이드바 → 알림 → 알림톡 템플릿 |
| 전체 알림 카탈로그 (38이벤트) | 사이드바 → 알림 → 알림 가이드 |
| BizM 콘솔 (외부) | 사장님 BizM 계정 직접 접속 |
