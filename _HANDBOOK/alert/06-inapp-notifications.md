# 인앱 알림 (앱 안 종 모양)

## 한 줄로 답하면
**사주플랜 앱 안의 알림 목록. 종 모양 클릭 → 알림 페이지. 영구 보존 (사용자가 직접 삭제). 알림톡과 별도지만 같은 정보 받음.**

## 인앱 vs 알림톡 차이

| 항목 | 인앱 | 알림톡 |
|---|---|---|
| 도달 | 사주플랜 앱 진입 후 | 즉시 (카톡) |
| 영구성 | 영구 (직접 삭제까지) | 카톡 30일 정도 |
| 차단 가능 | 사용자 불가 | 폰 차단 가능 |
| 채팅 중 | 발생 OK | 화이트리스트 외 차단 |

→ 알림톡 못 받아도 인앱은 살아있음 (정보 손실 없음).

## 이런 상황을 보셨나요?

### 회원이 "알림톡 안 왔어요" 라고 클레임

→ 인앱 (마이페이지 → 알림) 에 있는지 확인 권장. 같은 정보 받았을 가능성.

### "안 읽은 알림이 너무 많아요"

→ 인앱은 영구 보존이라 누적. 회원이 일괄 읽음 처리 또는 삭제 가능.

### "알림 클릭하니 빈 페이지로 가요"

→ deep link 어긋남. 옛 알림의 링크 페이지가 사라졌을 가능성 (예: 삭제된 게시글).

## 이건 정상인가요? 에러인가요?

| 상황 | 답변 |
|---|---|
| 회원 답변 받음 → 인앱 알림 + 알림톡 동시 | ✅ 정상 |
| 알림 클릭 → 해당 페이지 이동 | ✅ 정상 |
| 알림 읽음 처리 → 종 모양 빨간 점 사라짐 | ✅ 정상 |
| 채팅 중 → 알림톡은 차단, 인앱은 발생 | ✅ 정상 (정책) |
| 인앱 알림에도 정보 안 옴 | ❌ **사고**. 알림 시스템 점검 |

## 운영자가 확인하는 위치

| 확인 항목 | 메뉴 위치 |
|---|---|
| 회원 인앱 알림 (대신 조회) | 사이드바 → 회원현황 → 고객 리스트 → 상세 → 알림 탭 |

## 알림 화면 구성 (2026-06-11 개편)

"폰 아이콘엔 배지 2가 떴는데 앱을 열면 아무것도 없네?" 하던 혼란을 풀기 위해 **3단**으로 개편했어요.

### ① 새 알림 "마중" 배너 (홈 상단)
홈에 들어오면 **안 읽은 알림이 있을 때** 상단에 보라 그라데이션 배너 **"새 알림 N개가 도착했어요 ✨"** 가 스르륵 떠요 → 탭하면 알림함으로 바로 이동.
- 안 읽은 알림이 0이면 안 뜸 / X로 닫으면 그 세션 동안만 숨김
- 폰 배지(OS 푸시)와 앱 안 알림(인앱)을 이어주는 "여기 있어요" 안내 역할

### ② 알림함 (카드형 리스트)
각 알림이 흰 카드 + **카테고리 아이콘·색**으로 한눈에 구분돼요:
- 📢 전체공지(보라) · 👤 상담사(청록) · 👥 회원(파랑) · 🔔 개별(핑크)
- 안 읽은 건 또렷 + **빨간 점**, 읽은 건 흐림 + "읽음"

### ③ 알림 상세 (탭하면)
알림을 탭하면 **바로 링크로 튀지 않고** 상세 페이지에서 **제목·본문 전체를 먼저** 보여줘요. 이동 주소가 있으면 하단에 **[바로가기]** 버튼.

### 클릭하면 일어나는 일
1. 안 읽음 → **읽음 처리** (빨간 점 사라지고 마중배너 숫자도 줄어듦)
2. **알림 상세로 이동** (목록의 알림 내용을 그대로 넘겨 즉시 표시 — 재조회 없음)
3. 상세에서 **[바로가기]** → 해당 페이지로 (앱 안 화면이면 매끄럽게 이동 / 외부 링크면 외부 브라우저로)

> 데이터 출처는 그대로 `notification_log` 테이블. 디자인·동선만 개편했고 알림이 쌓이는 규칙은 동일.
