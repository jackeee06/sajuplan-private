# 알림톡 일괄 발송 (회원 그룹에게)

## 한 줄로 답하면
**운영자가 회원 N명 선택해서 일괄로 알림톡 보낼 수 있어요. 공지·이벤트 안내 용도. 마케팅성은 카카오 거부 가능.**

## 동작

```
운영자가 일괄 발송 페이지 진입
  ↓
대상 선택 (전체 / 필터 / mb_id 입력)
  ↓
템플릿 선택 (BizM 검수 통과한 것만)
  ↓
변수 입력 (예: 쿠폰명, 이벤트명)
  ↓
미리보기 → 확인
  ↓
백엔드가 각 회원에게 발송
  - 채팅 중 회원은 자동 차단 (정책)
  - 발송 결과 (성공 N건 / 실패 M건) 표시
```

## 이런 상황을 보셨나요?

### "이벤트 안내 1만명에게 보내고 싶어요"

→ 사이드바 → 알림 → 알림톡 발송 → 대상 선택 (전체 또는 필터) → 템플릿 선택 → 발송.

→ 주의: BizM 비용 (건당 약 7원) — 1만명 = 약 7만원. 사장님 BizM 잔액 확인 권장.

### "마케팅 알림 보내고 싶은데 거부됐어요"

→ 카카오 정책상 마케팅성 거부. 정보성으로 우회 (예: "이벤트 시작 안내" 같은 사실 위주).

### "발송 후 일부 실패"

→ 정상 가능. 회원 폰 차단 / BizM 응답 실패 등. 사이드바 → 알림 → 알림톡 발송 → 이력 에서 개별 결과 확인.

## 이건 정상인가요? 에러인가요?

| 상황 | 답변 |
|---|---|
| 일괄 발송 → 일부 실패 | ✅ 정상 (외부 사유) |
| 채팅 중 회원 차단 (recipient_in_chat) | ✅ 정상 (정책) |
| 발송 후 BizM 잔액 부족 → 중단 | ❌ 비용 부족. 사장님 BizM 잔액 충전 |
| 대용량 (10만+) 발송 timeout | ⚠️ 배치 분할 필요 (예: 1000명씩) |
| 마케팅 알림 시도 → 카카오 거부 | ✅ 정상 (정책) |

## 운영자가 확인하는 위치

| 확인 항목 | 메뉴 위치 |
|---|---|
| 일괄 발송 페이지 | 사이드바 → 알림 → 알림톡 발송 |
| 발송 결과 (개별 회원별) | 사이드바 → 알림 → 알림톡 발송 → 이력 |
| 사용 가능한 템플릿 | 사이드바 → 알림 → 알림톡 템플릿 |
