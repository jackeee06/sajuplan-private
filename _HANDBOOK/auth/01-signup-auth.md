# 회원 가입 / 로그인 / 비밀번호 찾기 / 소셜로그인

## 한 줄로 답하면
**phone 인증 + ID/비밀번호 또는 카카오·네이버 OAuth. 비밀번호 8자 + 영문/숫자 혼합 필수. 비번 찾기는 phone 인증 후 임시비번 카톡 발송.**

## 가입 방법 2가지

### 1. 일반 가입
- phone 입력 → SMS 인증번호 (5분 유효) → 확인
- ID, 비밀번호, 닉네임 입력
- 비밀번호: 8자 이상, 영문 + 숫자 혼합
- m2net 자동 등록 → 가입 완료

### 2. 카카오 / 네이버 OAuth
- "카카오 로그인" 또는 "네이버 로그인" 클릭
- OAuth 인증 → 프로필 정보 받음
- 사주플랜 가입 (소셜 매칭)
- phone 인증은 별도 필요 (m2net 등록용)
- **회원가입 코인(10,000) 지급은 로컬 가입과 동일** (2026-06-10 수정 — 아래 참고)

## 회원가입 코인 (가입 보상)

- 가입 완료 시 **회원가입 코인 10,000 즉시 지급** (`register_point`).
- **로컬(아이디 직접) 가입 · 소셜(카카오/네이버) 가입 모두 동일하게 지급된다.**
- 지급 금액은 관리자 설정값(`setting.member.register_point`) 기준 — 0 이면 미지급.
- **회원가입 쿠폰은 폐지됨** (2026-06-07). 쿠폰 대신 즉시 코인으로 통일 → 이중지급 방지.

> ⚠️ **2026-06-10 버그 수정**: 그 전까지 소셜(카카오/네이버) 가입자는 회원가입 코인을 한 푼도 못 받았다.
> (로컬 가입만 코인 지급, 소셜 가입은 폐지된 쿠폰 경로만 호출 → 쿠폰존도 비활성이라 0 지급)
> 수정 후 소셜 가입도 동일하게 10,000 코인 지급되며, 이미 가입한 소셜 회원 9명(카카오 3 + 네이버 6)에게 각 10,000 코인을 소급 지급 완료했다.

## 비밀번호 정책

- 최소 **8자**
- **영문 + 숫자 혼합** 필수
- 너무 단순한 비밀번호 (예: "12345678") 차단

→ 2026-05-25 정책 강화

## 비번 찾기 흐름

```
사용자가 "비밀번호 찾기" 클릭
  ↓
ID + phone 입력 → 매칭 검증
  ↓
phone 인증번호 발송 (5분 유효)
  ↓
인증 확인 → 임시비번 자동 생성
  ↓
알림톡으로 임시비번 전송
  ↓
사용자가 임시비번으로 로그인 → 비번 변경 권장
```

## 이런 상황을 보셨나요?

### 회원이 "인증번호가 안 와요" 라고 클레임

→ 가능 원인:
1. phone 차단 (070 또는 알 수 없는 번호)
2. BizM 서비스 일시 사고
3. 같은 번호로 5분 안 5회 초과 (한도)

→ 답변: "차단 설정 확인 + 5분 후 재시도 부탁드립니다."

### "비밀번호 8자 미만인데 가입 안 돼요"

→ 정책상 거부. 8자 이상 + 영문/숫자 혼합 안내.

### "카카오 로그인 했는데 phone 인증 또 해야 해요"

→ 정상. OAuth 만으로는 통화·채팅 불가 (m2net 등록 필요). phone 인증 별도 필요.

### "카카오/네이버로 가입했는데 회원가입 코인을 못 받았어요"

→ 2026-06-10 이전 가입자라면 발생 가능했던 버그(소셜 가입 코인 누락). 2026-06-10에 수정됐고, 기존 소셜 가입자 9명은 소급 지급 완료. 이후 신규 소셜 가입은 정상 지급. 그래도 누락 보이면 코인 내역 확인 후 운영자 수동 지급.

### "임시비번 받았는데 로그인 안 돼요"

→ 임시비번 = 한 번만 사용 + 일정 시간 (예: 30분) 만 유효. 만료됐을 가능성. 비번 찾기 재시도.

## 이건 정상인가요? 에러인가요?

| 상황 | 답변 |
|---|---|
| 5분 안 인증번호 5회 시도 → 6번째 차단 | ✅ 정상 (어뷰징 차단) |
| 비번 7자 가입 시도 → 거부 | ✅ 정상 (정책) |
| 카카오 가입 → social_provider 정상 매칭 | ✅ 정상 |
| 같은 카카오 계정 재가입 시도 → 차단 | ✅ 정상 (UNIQUE 제약) |
| 소셜 가입 후 회원가입 코인 10,000 지급 | ✅ 정상 (2026-06-10~ 로컬과 동일) |
| 2026-06-10 이전 소셜 가입자 코인 0 | ⚠️ 과거 버그. 9명 소급 지급 완료 |
| 가입 시 회원가입 쿠폰이 안 들어옴 | ✅ 정상 (쿠폰 폐지, 코인으로 통일) |
| 인증번호 발송했는데 안 오면서 비용은 빠짐 | ❌ **사고**. BizM/SMS 측 점검 |

## 운영자가 확인하는 위치

| 확인 항목 | 메뉴 위치 |
|---|---|
| 회원 가입 이력 | 사이드바 → 회원현황 → 고객 리스트 |
| 가입 사고 (m2net 등록 실패 등) | 시스템 로그 또는 사이드바 → 회원현황 → 고객 리스트 → 상세 |
| 임시비번 발송 이력 | 사이드바 → 알림 → 알림톡 발송 |
