💬 카카오톡에서 보고 계신 경우
오른쪽 위 ⋮ (점 세 개) 메뉴 → 「다른 브라우저로 열기」 선택 후 다운로드 받으세요.

설치 시 "악성 앱 의심" 메시지가 뜨면

방법 1 다운로드 위치를 바꾸세요 (가장 쉬움)

  1. 다운로드 받은 파일을 「내 파일」 앱(파일 관리자)에서 열기
  2. 파일 길게 누름 → 「이동」 선택
  3. 「내장 저장공간」 의 가장 위(루트)로 이동
  4. 옮긴 파일을 누르면 그냥 설치됨

💡 다운로드 폴더의 파일은 보안 검사 대상이지만, 루트의 파일은 검사 안 받음

방법 2 보안 기능을 잠깐 끄세요

  1. 스마트폰 「설정」 앱 열기
  2. 「보안 및 개인정보 보호」 누르기
  3. 「자동 차단 기능」 또는 「Auto Blocker」 찾기
  4. 일시적으로 끄기 → 앱 설치 → 다시 켜기

💡 설치 끝나면 다시 켜도 이미 설치된 앱은 영향 없음

자주 묻는 질문

Q. 정말 안전한가요?

네, 사주플랜이 직접 만든 정식 앱입니다. Play Store 정식 등록 전이라 일부 보안 기능이 보수적으로 차단할 뿐, 앱 자체는 안전합니다.

Q. iPhone 에서도 받을 수 있나요?

현재는 안드로이드만 지원합니다. iOS 는 App Store 정식 출시 후 이용 가능합니다.
