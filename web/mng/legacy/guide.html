Charts (ApexCharts)

대시보드에서 ApexCharts CDN을 사용합니다. Area, Donut 차트 예시는 대시보드에서 확인하세요.

<script src="https://cdn.jsdelivr.net/npm/apexcharts"></script> // Area Chart 기본 패턴 const chart = new ApexCharts(el, { chart: { type: 'area', height: 280, toolbar: { show: false } }, colors: ['#465fff', '#34d399'], stroke: { width: 2, curve: 'smooth' }, fill: { type: 'gradient', gradient: { opacityFrom: 0.35, opacityTo: 0.05 } }, // 다크모드 반응: tooltip: { theme: isDark() ? 'dark' : 'light' } });
